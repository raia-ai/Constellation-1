# Creating Transactions via Transaction Wizard

This document will reference the Transaction Wizard, how to enter a transaction through its step-by-step instructions and how to disengage the Wizard should you wish to continue entering transactions manually

Note: to turn off the Transaction Wizard at any time:

* Select System Settings from left hand navigation bar
* Select Advanced Settings
* Under User Preferences, toggle “Enable Transaction Wizard” off

To add a transaction when Wizard is engaged:

Navigate to Transactions in left hand navigation bar

From Transactions list, select Add+ button and the Transaction Wizard screen will appear and guide you through the entry process, in particular:

* What side of the transaction are you on? Buying, selling or both
* Entry of Buyer and seller-please note that Transaction Wizard only allows entry of 1 buyer and 1 seller-additional buyers and sellers can be added after transaction is saved as noted below
* Note that when using the Transaction Wizard to create a transaction, existing contacts will be added to the transaction as either buyer or seller. To change the contact type, ie. changing Buyer to Tenant or Seller to Landlord, they can be edited after transaction is saved as noted below

Note: if buyer or seller is not available in the drop-down list, select + Add new and enter contact into system

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/a5132ba4-a024-496d-9929-19a96bd51e8e)

Enter transaction details including all mandatory fields, noted by an asterisk \* at end of Field Label name

_**Please note: The State/Province will auto-populate with the same information provided in the Company profile. For further information, please click here:**_ [**Constellation1 Commissions Online - Company Profile Maintenance**](https://constellation1.na3.teamsupport.com/knowledgeBase/16981163)&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/5b05e40a-f305-4c3a-bfbc-cf794ac0a0a5)

Select Next and transaction will be saved in system.  You will then be prompted with the following options:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/973dde3d-82e1-4e07-bd51-a9c71c376414)

* View Detail-to view previously entered details on transaction or to add additional items such as Additional details, Deposits etc
* Edit Contact-in order to add additional or edit previously entered contact details on transaction such as Buyer to Tenant, Seller to Landlord etc
* New Transaction-in order to add another transaction to system
* Add Commission-in order to add commission details to transaction
* Disable Wizard-should you wish to disable the Wizard at this time
* Add Commission-allows you to add commission details to transaction

Add Commission

By selecting Add Commission as noted above, the system will then take you to the Transaction Commission tab

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/d8ea7c82-8c68-40e2-9731-5d7c179ddc95)

The system will automatically pre-fill the Sale Price entered previously and may be changed at any time from this tab

Either select a Commission rule from the dropdown list to automatically calculate commission or manually enter the gross commission as a dollar amount or percentage by toggling the indicated button above

If desired, under Additional Detail, the following options may be selected

* If your office has multiple General Ledger profiles established, you may select the appropriate GL profile for this transaction
* Force Validation on Commissions will automatically compare commissions expected on transaction against commissions assigned (including partial payment transactions) to ensure all funds are accounted for
* Commission Disbursed by Third Party-should the funds be disbursed by escrow/title company/attorney etc
  * If "Commission Disbursed by third party" toggle is engaged:
  *   Complete Commission Disbursed section if appropriate selecting Disbursing party from list and assigning the Disbursement Form.  The assigned Disbursement Form can be edited by selecting the edit tool and adding or deleting payment options by toggling these items on or off

      ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/1b73bc9a-7e14-48c6-acd5-a5689b267681)

You will then assign the Selling and Buying side commission on deal and assign an agent either from within your office by selecting their name from the dropdown list if your office is on that side, or you can add a cooperating agent to the deal by selecting Add Cooperating Agent and adding from dropdown list if they are outside your office![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/feaa53e1-20c8-490b-85f2-4ffa07055198)

Based on Commission Plan assigned to agent in Agent Profile, the agent and company portions of commission will automatically calculate.

Select Save and transaction will be saved with updates
