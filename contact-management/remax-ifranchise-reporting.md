
# RE/MAX iFranchise Reporting | Constellation1 Customer Hub

Constellation1 Commissions supports reporting of transactions to RE/MAX Integra via the iFranchise system.

*   To locate the batch file navigate to:
*   Reports > Franchise Reports > RE/MAX Reports > Monthly Production Summary (iFranchise)

 ![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-REMAXiFranchiseReporting/ifranchisereportin1.png)

*   Select the desired parameters.
*   This report is exportable in .txt file format as supported by RE/MAX Integra which can then be uploaded directly to iFranchise:

 ![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-REMAXiFranchiseReporting/ifranchisereportin2.png)

*   In addition to the file upload, RE/MAX franchisees will be required to manually update the following Business Activity Summary data points within the iFranchise portal:
*   Total Volume for the month  
    Total GCI for the month  
    Total Ends (Units) for the month
*   For questions on the import process please contact [support@constellationreg.com](mailto:support@constellationreg.com%20) or RE/MAX Integra support.
