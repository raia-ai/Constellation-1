# Contact Tab of the Transaction

This is article will assist you with a better way to track your business. By adding Mortgage Companies, Pest Control Companies, Home Inspectors, Home Warranty Companies, and Title/Escrow Companies. If you are in an escrow state you can also utilize this tab to track Escrow once the Escrow Title Resource has been selected the screen then changes to allow for the Escrow number to be entered. There is also a corresponding report under Transaction Reports, and Transactions by Contacts.

*   Navigate to the Large Back Plus icon to expand the tabs.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-ContactTabTransaction/contacttabtransac1.png)

*   On the Contacts Tab, to add a contact you must first select the the small plus sign icon just as if you are adding anything new to the transaction itself.
*   From there, scroll down to see the drop down for the the various resources now available on this tab.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-ContactTabTransaction/contacttabtransac2.png)

*   If Escrow Title Companies is selected the screen will then change to allow to enter the Escrow number.
*   If using the Escrow/Title Companies and wishing to now run the report to track your Escrows please navigate to the Reports, Transaction Reports, Report by Contacts, and select the report type of Escrow/Title.
