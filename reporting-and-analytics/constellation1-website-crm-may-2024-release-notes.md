## Constellation1 Websites Release Details

New! State Landing Pages  

The website now allows administrators to customize state landing pages. A new "State Landing Page" option has been added to the Page section in the Layouts.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-ReleaseNotes/Website%20CRM/2024/05%20May/2024-April-Website-StateLanding.jpg)

This update enables administrators to customize state landing pages by adding unique content, page titles, keywords, and descriptions to optimize each page for search engines. 

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-ReleaseNotes/Website%20CRM/2024/05%20May/2024-May-Website-StateLandingPageEdit.jpg)

Content can be added to the top and bottom sections of the state landing pages.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-ReleaseNotes/Website%20CRM/2024/05%20May/2024-May-Website-StateLandingPage.jpg)

Example of a state landing page,   
[https://showcase.rdeskbw-stage.com/homes-for-sale/CA/](https://showcase.rdeskbw-stage.com/homes-for-sale/CA/)

The new state landing page layouts does not affect any customized state landing pages. 

## Bug Fixes

**Website**

1\. Fixed an issue with the listing summary report not showing the data charts. 

Previously, the charts were not showing on the listing summary report.

Now, the charts will show on the report as expected. This issue has been resolved. 

2\. Fixed an issue when adding a Friendly URL to the website admin. 

Previously, adding a Friendly URL in the website admin would trigger an error. 

Now, this issue has now been fixed. Friendly URLs can be added without errors.

## Technology Updates 

**Websites**  
1\. We’ve updated elastic search from v7.0 to v8.0.
