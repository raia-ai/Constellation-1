# Setting up Withholdings or Garnishments

This article will outline how to set up an Withholding for an Agent to be automatically taken from each transaction until such time as the cap has been met or time frame indicated.

*   Navigate to the Withholdings tab of the Agent's profile.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/WH.png)

*   Select the Resource type, and the Resource name via the magnifying glass icon.
*   Will this deduction still be a part of their reported income? If yes check the Apply to T4A/1099
*   Reference ID if applicable
*   Either a Percentage or Flat Amount
*   Is there a cap? Enter the cap amount and if an end day applies.
*   Hit save
*   If any open transactions are pending for this agent be sure to re fresh them and ensure that the Withholdings is visible in the Agent Detail Window of a transaction.
