# Generate instance-wide reports using a Site Admin account in Transactions

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/a35e45e5-ec1b-45b1-896d-e48fb90dc65e)

*   Log in to the Site Admin account.
*   Click on the "Reports" tab.
*   Click on "Branch Results" under "Transactions".
*   Set-up the search filters as shown below and click on "Go":![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/ebbd9469-a387-4664-96f7-214bf8a16dc8)

*   Click on "Print" and set the Destination as "Save as PDF" as shown below:
*   ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/ff5d5621-540f-4170-9a8d-69a67b8db80d)

*   This will generate a PDF file with all of the transactions information
