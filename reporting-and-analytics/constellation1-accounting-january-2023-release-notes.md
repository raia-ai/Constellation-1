# Constellation1 Accounting - January 2023 Release Notes

Sales -

*   **Commission Statement**:  When system setting **1CHKPERTRN** is disabled, and system setting **COMMISSIONSTMT** has **Text Value** = **Legacy**, if all details of a sale will not fit in the remaining space on the page, the details of that sale will start on a new page.

Agents -

*   The format of the 1099-MISC and 1099-NEC forms has been updated to support the IRS' changes for 2022.
*   **Incentive Level** Report - **Office Totals** and **Grand Totals** have been added to the **Basis Amount** and **$ to Next Level** columns.  

Bank - 

*   A **Payroll Check Register** has been added to the reports navigation menu.
    

Payroll -

*   Voiding of payroll payments will no longer produce an error.
*   In the payroll reports navigation menu, the title **Form 941** has been changed to **Form 941 (unofficial)**.
*   A **Payroll Check Register** has been added to the reports navigation menu in the **Bank** module.
