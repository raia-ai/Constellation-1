# Using Report Scheduler | Constellation1 Customer Hub

Commissions Online has the ability to create a schedule to generate and email specific reports at different times automatically. 

Select **Reports** on the left navigation bar, and navigate to Report Scheduler tab

Select **Add+** button

There are currently four reports that can be scheduled;

*   Agent Incentive Level
*   Agent Production Ranking
*   Cash Flow Projection
*   Transaction Commission Detail

To create a schedule select **Add**:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/c722944a-0de4-403e-8629-29796efa6cbe)

Complete the following:

*   **Schedule Name;** assign a name to describe the report you want to schedule
*   **Report;** select the desired report you wish to schedule
*   **Frequency;** select Daily, Weekly or Monthly. 
*   **Time;** select the time for the report to send if frequency is Daily. Frequency options for Weekly and Monthly will provide days of week OR days of month in addition to a time. 
*   **Time Zone;** will auto-populate based on your location, but can be changed if you prefer. 
*   **File Format;** select PDF or XSLX
*   **Recipients;** select the applicable non-agent user whom you wish to receive the report

**Filters** allow you to filter by Offices, Agents, Transaction Status, etc.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/f9bf5a9e-2437-4f64-93f7-1dbe9649a4d0)By expanding the **Advanced Filters** or **Formatting** sections you have more options to select based on options available on that specific report. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/47765556-3b5c-4532-96b3-39b0d9f53d5b)

Once you are finished, select **Save** 

The report will be generated and emailed to the Recipient(s) as an attachment in the format specified based on the settings saved. 

To pause a scheduled report, toggle ON the **Pause Schedule** toggle, then select **Save** button. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/4058ad27-bae8-4b52-9e19-26b4c20de1b6)

To edit, copy or delete a scheduled report, from the Report Scheduler landing page, select the ellipsis to the right of the Scheduled report and make your selection

*   **Edit** will allow you to make changes to a pre-set schedule
*   **Delete** will delete the schedule
*   **Copy** allows you to copy the contents of a pre-formatted schedule so it can be modified and renamed without having to create a new schedule from scratch
