
# REALedger - How to Create 1099 Electronic Corrections File

The purpose of this article is to provide the steps needed to create a 1099 electronic corrections file to upload to the IRS.

To create an electronic corrections file, navigate to Agents>>Reports>>Agents Earnings and Income>>1099s.

*   Select **Electronic Corrections** from the **Type of Data to Include** dropdown

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/1d270ab2-b304-42de-90b9-4303763bfc09.png)

*   Print Pre-1099 Report

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/693599dd-afb9-4465-8a63-b98329ec86d2.png)

After closing the Pre-1099  Report, **Correction** will appear in the **Original, Replacement or Correction File?** dropdown.

*   Click **Generate Magnetic Media** button

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/8d90f43e-78ae-43cf-9db6-ad99b32491e9.png)

*   Click **Yes** to the pop-up

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/fb7a6bbe-3f0b-4dde-a116-24eeb12b32f8.png)

*   Make note of the file location 
*   Click **OK**

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/5809c3b9-fe5d-40c1-847f-60678005c499.png)

*   Locate the file and upload to the IRS
