# Paradym by Constellation1 December 2024 Release Notes

Production Release December 2024

## Release Summary

**Replace Posts:** Updates to replacing posts on social media. \
**Bug Fixes**

## Release Details&#x20;

Replace Posts\
We've updated the ability to replace social media posts. When users select a video post to replace an existing post, the video will automatically play in the preview window, enabling them to view it before confirming the replacement.&#x20;

Bug Fixes&#x20;

1\. We fixed an issue with a user establishing a connecting to LinkedIn.&#x20;

Previously, when a user tried to connect their Paradym profile to LinkedIn, they were being redirected to a 403 error message.

Now, this issue has been resolved.&#x20;

2\. We fixed an issue with social postcards showing with a pending status on the Social Analytics page, when it has been replaced.

Previously, when a social postcard was replaced, the original postcard would continue to display a pending status on the Social Analytics page, despite the replacement.

Now, this issue has been resolved.&#x20;

3\. We fixed an issue with the CMA Agent Report connection status for agents.&#x20;

Previously, The CMA Agent Report displayed an incorrect connection status for agents listed in the report.

Now, this issue has been resolved.&#x20;

4\. We fixed an issue with an error message showing when brokers used the Agent Look-Up tool.

Previously, when a broker used their back office to look up users and agents, they would get an error after viewing multiple pages.&#x20;

Now, this issue has been resolved.
