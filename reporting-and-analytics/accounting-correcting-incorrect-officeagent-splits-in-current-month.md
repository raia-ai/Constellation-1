# Accounting - Correcting Incorrect Office/Agent Splits in Current Month

The purpose of this article details the steps to correct a sale transaction if agent was over/underpaid in the current month.

**Agent Overpaid in Current Month**

*   Navigate to **Sales > Processing > Sales Transactions**
*   Select **Closed** radio button
*   Find the sale in the left pane, highlight the sale and select **Edit**
*   Click on **Commission** tab
*   In the **Agent Commission** section, select the green plus sign, select the agent from pop-up list, and select **OK**.
*   Enter amount the agent was overpaid as a positive value in the **Office$** field, this will autofill the **Agent$** field with the amount they were overpaid with a negative value, equal to the amount they were overpaid.
*   The post-split deduction for agent payments will autofill with negative values, equal to the amount the agent was overpaid and the **Agent Payment** will be zero.
*   Uncheck the **Recalc at closing** box and select **Save**.
*   The record should look like the screenshot below.

 ![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/727.png)

*   Select **Actions** button
*   Select **Issue Payments**

**![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/728.png)**

*   Click **Select**
*   Click the **Re-calculate Splits and Agent Charges**
*   Click **Pre-Payment Report**
*   Click **Commission Statement**
*   Click **E-mail Commission Statement**, if you use this feature.
*   Click **Process**
*   Click **OK** to the **Issue Commission Checks** pop-up.
*   Close the **Issue Commission Payments** tab.
*   An agent charge will be created to be repaid by the agent, either in a future closing or they can issue you a check and you can enter that as a non-commission agent payment, in the bank module.

**Agent Underpaid in Current Month**

*   Navigate to **Sales > Processing > Sales Transactions**
*   Select **Closed** radio button
*   Find the sale in the left pane, highlight the sale and select **Edit**
*   Click on **Commission** tab
*   In the **Agent Commission** section, select the green plus sign, select the agent from pop-up list, and select **OK**.
*   Enter amount the agent was underpaid as a positive value in the **Agent$** field, this will autofill the **Office$** field with the amount they were underpaid with a negative value, equal to the amount they were underpaid.
*   The **Agent Payment** field will autofill with the amount the agent was underpaid.
*   Uncheck the **Recalc at closing** box and select **Save**

**The record should look like the screenshot below.** 

**![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/729.png)**

*   Select **Actions** button
*   Select **Issue Payments**

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/730.png)

*   Click **Select**
*   Click the **Re-calculate Splits and Agent Charges**
*   Click **Pre-Payment Report**
*   Click **Commission Statement**
*   Click **E-mail Commission Statement**, if you use this feature.
*   Click **Print Checks** or **ACH Advises** button, whichever is applicable.
*   Click **Process** button.
*   Close the **Issue Commission Payments** tab.
