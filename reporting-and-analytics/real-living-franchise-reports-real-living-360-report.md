# Real Living Franchise Reports (Real Living 360 Report)

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-RealLivingFranchiseReports/RealLivingFranchiseReports1.png)

This article outlines critical information for setting up the Back Office system to support reporting requirements for Real Living franchisees (an affiliate of Berkshire Hathaway HomeServices).

*   For the report to successfully display correct information, users will need to configure transaction types.
*   From the Set Up Menu> Table Lookups > select **Trade Codes**
*   Select the Franchise Transaction Type.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-RealLivingFranchiseReports/RealLivingFranchiseReports2.png)

On the Main tab of the transaction you will also select Transaction Group of either Residential of Commercial. 

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-RealLivingFranchiseReports/RealLivingFranchiseReports3.png)

*   To Generate the Reports navigate to Reports > Franchise Reports > **Real Living**
*   There are currently two reports available in this folder: a monthly production and the Real Living 360 report used for surveys on all recent closings. The latter can be exported to .csv format for direct import to Real Living's third party partner provider.
