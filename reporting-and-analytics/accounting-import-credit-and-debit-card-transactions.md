# Accounting - Import Credit and Debit Card Transactions

The purpose of this article is to provide instructions on how to use the **Import Credit and Debit Card Transactions** functionality.  This process is used to import credit card and/or debit card transactions from your bank. 

**Instructions for Importing Credit Card Activity:**

*   Log into bank and download applicable credit card account activity, using file type QFX.
*   Navigate to **Accounts Payable > Processing > Import Credit and Debit Card Transactions**.  
*   Select **QFX File Type Credit Card.**
*   Select **Invoice Date**, or accept default date.
*   Select **Due Date**, or accept default date.
*   Select **Class,** or accept default class.
*   Select **Resp. Code**, or accept default class.
*   Select credit card provider from the **Vendor** dropdown.
*   Click **Browse** to navigate to the downloaded file from the bank. 
*   Click **Open File**.
*   Select **Bill Type** from the dropdown.
    *   Select **Unpaid**, if you plan to issue a check to the credit card company.
    *   Select **Manual**, if you plan to pay online.

From the resulting **Card Activity** grid, delete and/or edit the line items as needed.

**To Delete Line Items:**

*   Put a check mark to left of line items you would like to remove. 
*   Click red "x" to remove applicable line items.

**To Edit Line Items:**

*   Select **Account** from dropdown, in the **Card Activity** grid.
*   Select **Office** from dropdown, in the **Card Activity** grid.
*   Select **Project** from dropdown, if applicable, in the **Card Activity** grid.

**To Create Bills:**

*   Click **Create Bills**.
*   Click **Yes** to the following pop-up to continue.
    *   Click **No** if you do not want to continue. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/2c226260-fd97-482b-a2a5-5d7040d1a7bc)

If **Yes** is selected, click **OK** to the following pop-up:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/8b2dccb8-5549-4442-b80c-e5a80a483ebc)

*   Navigate to **Accounts Payable > Processing > Bills**.
*   Make edits to **Unpaid** or **Manual** bill created, if applicable.  
    *   If **Unpaid** bill is created, navigate to **Accounts Payable > Processing > Pay Bills**, when ready to print check for the bank.  

**Instructions for Importing Debit Card Activity:**

*   Log into bank and download applicable debit card account activity, using file type QFX.
*   Navigate to **Accounts Payable > Processing > Import Credit and Debit Card Transactions**.  
*   Select **QFX File Type Debit Card**
*   Select **Invoice Date**, or accept default date.
*   Select **Class,** or accept default class.
*   Select **Resp. Code**, or accept default class.
*   Click **Browse** to navigate to the downloaded file from the bank. 
*   Click **Open File**.

From the resulting **Card Activity** grid, delete and/or edit the line items as needed.

**To Delete Line Items:**

*   Put a check mark to left of line items you would like to remove. 
*   Click red "x" to remove the unneeded line items.

**To Edit Line Items:**

*   Select **Vendor** from the dropdown, in the **Card Activity** grid.
*   Select **Accoun**t from dropdown, in the **Card Activity** grid.
*   Select **Office** from dropdown, in the **Card Activity** grid.
*   Select **Project** from dropdown, if applicable, in the **Card Activity** grid. 

**To Create Bills:**

*   Click **Create Bills**.
*   Click **Yes** to the following pop-up to continue.
    *   Click **No** if you do not want to continue. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/f08d3980-ff3f-4c02-9218-3d67f86e05aa)

If **Yes** is selected, click **OK** to the following pop-up:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/8b2dccb8-5549-4442-b80c-e5a80a483ebc)

*   Navigate to **Accounts Payable > Processing > Bills** > **Manual** view.
*   Make edits to **Manual** bill created, if applicable.
