
# Creating Custom Reports | Constellation1 Customer Hub

Commissions Online provides a number of predefined reports however, on occasion users want to create their own reports. 

Commissions Online provides a few predefined reports that can be copied and further customized for your business requirements in addition to the ability to create your own report from scratch. 

**Customizing a Predefined report**

The following predefined reports can be copied and further customized by adding/deleting report fields, moving of columns etc. 

*   Agent Income Summary
*   Transaction Commission Detail
*   Agent Transaction Detail
*   Pending Transactions

To copy a predefined report, from the left navigation bar select **Reports** and select one of the reports listed above.

Select **Copy** button

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/3feb5ae9-4cc0-41c3-8fd6-40e144cf09bb)

The application will create copy of the report including all report fields, filters, advanced filters and formatting from the original report. This new custom report can be found on the Custom tab of the Reports menu.  At this point you can:

*   Provide a name and description of this custom report
*   Add/edit any filters, advanced filters or formatting options  
*   Remove report fields by clicking on the red X in the right corner of the field you want to delete
*   Add any report data field from the list on the left, then drag/drop to the Preview Panel section 
*   Rearrange the order of the fields displayed on the report by selecting a field box and physically moving it

Once the customization is complete, select **Save/Save as New**

**Please note that you can have a maximum of 15 records available on reports**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/150d2301-7933-4f3d-bd15-27b76aee20d7)

As mentioned above, all custom reports will be found on the Custom Reports tab in the Reports menu.

To delete a custom report, from the left navigation bar, select **Reports / Custom Reports.** Select the applicable report and select the delete icon at the top right of the report. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/85161a7d-280a-403b-86c2-c01dd021b563)

**Creating a Custom Report from Scratch**

From the Custom Reports tab, select Add +

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/0beb83db-6c62-416d-b844-ca0a21ac122a)

A window comparable to below will be displayed. At this point you can:

*   Provide a name and description of this custom report
*   Add/edit any filters, advanced filters or formatting options
*   Add any report data field from the list on the left, then drag/drop to the Preview Panel section
*   Remove report fields by clicking on the red X in the right corner of the field you want to delete
*   Rearrange the order of the fields displayed on the report by selecting a field box and physically moving it

Once the customization is complete, select **Save/Save as New**

As mentioned above, all custom reports will be found on the Custom Reports tab in the Reports menu.
