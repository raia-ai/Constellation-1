# Process T4As when Integrating with Commissions Desktop

The purpose of this article is to provide the information to Process T4A's.

**Prior to processing T4Aâ€™s we strongly recommend all mandatory information is entered correctly:**

* Each Agent should be configured with their correct legal name and Social Insurance Number/Business Number. If this information is missing the application will display a warning "There is an invalid SIN or Payroll number for affected Agents. Please go to the Agent profile to correct them first."  Agents will also require the usual mandatory information such as a complete address, province and postal code.&#x20;
* We recommend confirming values recorded on the T4A form by using the Transaction Commission/Fee/Sales report via Reports / Transaction Reports
* Please configure the report as noted below:
  * Date Type = Process Date
  * Report Type = Detail
  * From Date = January 1 for the previous calendar year
  * To Date = December 31 for the previous calendar year
  * Include Closed should be checked&#x20;

_Tax Year Option should be changed via the Additional Options panel if there are Agents who deferred commission to another tax year and select Page Break Between Agents so users can provide the report along with the T4A._

_If you have entered any history for Agents that have changed their registration to transact as a Personal Real Estate Corporation, please ensure you select Actual Commissions. This will eliminate any history that may have been entered to capture commissions earned as an individual vs a Personal Real Estate Corporation towards existing commission plan mid-calendar year. The reported amount may require adjustments (+/-) to the T4A amounts when reviewing as described below._&#x20;

**NOTE: Effective Tax Year 2023, Box 015 is only mandatory if there is an amount in Box 016. Here is the link from Canada Revenue Agency site;** [**https://www.canada.ca/en/revenue-agency/services/tax/businesses/topics/payroll/completing-filing-information-returns/t4a-information-payers/t4a-slip.html**](https://www.canada.ca/en/revenue-agency/services/tax/businesses/topics/payroll/completing-filing-information-returns/t4a-information-payers/t4a-slip.html)

_**Instructions to Print and E-file (Access Code from Canada Revenue Agency is required to E-file):**_

**1.** Navigate to Accounting menu and select Process T4As.

**2.** You must Review the information prior to generating the T4As. Click "Select All" radio button to select all Agents or place a check to the left of each Agents name.

**3.** Click **Preview** (eye icon) at top right of Process T4As window

**4.** Agent T4A window displays the amounts, including any Agent's history as entered on the Summary tab of their profile.  You can make adjustments if needed.  The Adjustment field supports both a positive or negative integer (to reduce an amount), if you feel that the values are incorrect. Values displayed as Other Income (Box 28) represent the Manager Override/Recruiter/Mentor income that the Agent has earned in the previous calendar year. If multiple Agents were selected, click "Next icon" at the top right of window or "Ok icon" when you are finished.

**5.** Commissions will record a Reviewed Date which allows users to continue by selecting "Print Slips (printer icon)" or "E-file (plane icon)" buttons for selected Agents.

**6.** When "E-file" is selected, you will be presented with a submission window, complete the mandatory fields as required and click Send. The system will indicate where the "xml" file is saved, please ensure that you make note of its location so that you can submit the appropriate file once you have been redirected to the Canada Revenue Agency (CRA) site.

* When user selects OK, the page will be redirected to the CRA website.  Click Cancel to go back to the previous screen.
* If ready to E-file, click the button labelled, "Enter Internet file transfer (XML) secure Web site".
* User will agree to the Canada Revenue Agency terms at the very bottom of the page.

**Follow the directions as provided by CRA site with your Account Number (this is usually your Payroll Number, ex. 123456789RP0001) and (Web) Access Code available.**

* When the process is completed, E-File Submission ID for the applicable Agents will be displayed.
* Once filed and if user needs to make an adjustment, user must meet CRA guidelines as posted on the CRA website before attempting to re-process.

**7.** To print slips, simply select the appropriate Agent(s) and the information will populate with the values on the T4A slip.  A T4A summary will also be included in ".pdf" format.
