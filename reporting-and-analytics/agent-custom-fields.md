# Agent Custom Fields | Constellation1 Customer Hub

In the event your office wishes to track data for your agents that doesn't have an existing data field created, you can now create and track your own which can then be added to any Custom Report or template you desire.

In order to access **Custom Fields**, from the left navigation bar, navigate to System Settings/Advanced Settings and click on the **Custom Fields** tab. A list of existing Custom Fields will appear.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/df69cbe0-88b0-4025-aa8d-793bde6fa657)

**Adding a Custom Field**

In order to add a new field select **+Add Custom Field**

Assign a name to data field in the Field Name box and choose the type of data field you would like to add

**Text** will allow you to enter a free form field of your own choosing on the fly

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/6e7e65ed-0cc1-4351-9b5a-9ed3c32f59bd)

**Drop down** will display an additional option to "hard code" a list of Allowed Values that may be selected from when adding the field to an agent by entering them in the "Allowed Values" section one at a time.  Should you wish to add more options to this dropdown list, simply select **Add Allowed Value** and enter another option to the list.

Should you wish to delete one of these items from the list, simply click on the **delete** button and the option will be removed

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/130a34db-d7bf-4688-8735-337c7d616cd8)

**Editing Custom Fields**

From the main list of Custom Fields, select Action to the right of the field in question.  The field can be deleted by selecting **Delete**.

By selecting **Edit**, the field configuration will open at which point in time it can be edited as desired

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/67e82129-6224-4c52-9368-8f486cdc661d)

**Fields appearing in Agent Profile**

Once a custom field has been created, an additional drop down section named Custom Fields will appear in the Agent profile which will allow you to start tracking that new data field

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/142e1e30-137a-4283-8237-44350ac1f4ac)
