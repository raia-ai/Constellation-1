# How to View and/or Create an Action Plan

**To view an existing Action Plan, click on the link of the title of the Action Plan.** This will allow you to look at the Action Plan you wish to see.

**![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/9ea4035a-4962-4bbd-8fe8-a0d57cd538eb)**

**To create a new Action Plan, select the +Create New Action Plan button.**

**![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/039e3a64-bbdc-481a-aeaa-bea55bf7d0db)**

The New Plan overlay will appear. Choose a schedule type by clicking on one of the three options.

**![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/ffa7f3da-2182-4b35-8260-7724525b32e3)**

**Enter a plan name in the overlay.** This will be the title of the Action Plan that displays in your grid.

**![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/e64fefb7-daca-4f85-a8f6-61922142721f)**

Clicking Cancel will take you back to the Action Plan Manager view of all existing plans.

Click the Save button. A confirmation message will appear indicating that you have successfully created the new plan.

If you have issues with this or with any aspect of the Nestfully product, please contact [support@nestfully.com](mailto:support@nestfully.com).
