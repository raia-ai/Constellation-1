# Transactions - Documents Tab | Constellation1 Customer Hub

This article will provide you with information regarding the Documents tab of the Transaction and its use.&#x20;

Each Commissions Online account provides 5 GB of storage. If you require more, please email [commissions@constellation1.com](mailto:sales@constellation1.com) and a member of our Team will be happy to assist.&#x20;

Documents support MS Word, MS Excel, PDF, txt, etc. type files and can be added to Pending, Canceled or Closed Transactions.

**Add a Document**

To add a new document to a transaction, navigate to the Documents tab and select ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/83113d02-c1ba-4530-9c17-4f0e04f37b88) at the top right.

Select Choose File and navigate to the location of the document.&#x20;

Enter a Description, toggle ON Private if you do not want this document shared with other Master Users, Management Users, Office Administrators and/or the Agent and select Save.&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/4a933ff8-6ead-40a5-9114-beb54c6fd6df)

Once the document is added, you can Edit, Delete, Mark as Private (if Private not toggled on previously) and Download. &#x20;

Edit allows you to change the Description only.  If you added the incorrect document, you will select Delete and add again.&#x20;

Download will save a copy of the document to your PC's local download folder.&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/3e862e87-1839-4c5d-af88-0e95b6103de0.png)

By design, all Closed transactions will automatically add the Review Closing Report which will not be visible under the Agent and/or Team Leader views. Please click here for information relating to the [Agent and Team Leader View of Program.](https://constellation1.na3.teamsupport.com/knowledgeBase/20920481)

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/0200fd90-f991-4705-b831-0e7ad33d7492)

If you wish to monitor your Document Storage; on the left navigation bar select **System Settings / Company**.
