# How to Edit an Existing Action Plan

To view an existing Action Plan, click on the link of the title of the Action Plan.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/68eb8d58-5859-4d2a-a42d-4f65aa196498)

To edit an existing Action Plan, click the **edit** (![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/6ac84518-3d38-4b16-b270-6d727ff88eaf)) icon and the Action Plan details will display.

Here, you'll be able to edit the title, add/edit/remove activities, and add/edit/remove recipients.

Click **Save** OR click the **Preview** (![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/c75a2d88-d455-45d4-af82-c15cfe0cd9f5)) button to preview the whole plan and to view the changes made. You can use the arrow to scroll through each activity.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/3eee87bd-ca5b-4259-933a-1951cf236354)

You can also click **Print** to a copy of your all activities in your Action Plan.
