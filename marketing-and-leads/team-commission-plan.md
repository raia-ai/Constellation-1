
# Team Commission Plan | Constellation1 Customer Hub

This article details the steps required to configure a Team Commission Plan.

On the left navigation bar select **Commissions** / Commission Plan tab and select **Add.**

Complete the details as required:

*   Plan Name - the unique name for this Team Commission Plan
*   Plan Type - Flat Rate, Sliding Scale or Partial Payments; [Creating Flat Rate Commission Plans](https://constellation1.na3.teamsupport.com/knowledgeBase/23836541)
*   Calculate Agent History Using - select from drop down
*   Period Starting From - select from drop down
*   Period Ending - select from drop down
*   Commission Basis Date - select from drop down
*   Include Pending Transactions? - toggle ON if applicable
*   Include Non-Commission Income - toggle ON if applicable

Expand Advanced Options and ensure Is This Team Plan? is toggled ON.Â 

Calculate Adjusted Company Portion on, select from drop down as applicable

Enter the Ceiling Amount for ALL members to contribute towards the next Tier whether all members of the team or only one member of team is attached to a particular transaction.

  
If user needs to attach any deductions to the Team Commission Plan, select the +Add Deduction link and complete as needed.

Additional Notes:

*   The plan should be assigned to all members of the Team via Agents / Commission Details tab / Teams / Default Commission Plan
*   The plan will calculate the commission using the Team Leaders' Anniversary timeline
*   More information about Teams can be found here; [Teams](https://constellation1.na3.teamsupport.com/knowledgeBase/17481761)
