# Nestfully - How to refine your search using Keywords?

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/e0ddbcfe-d295-4f68-8a62-4b5e9dbad96b)

Keywords can help you refine searches when the property that you are searching for does not show up on the initial search.  
Keywords search the Description of properties to limit the search results to properties containing that keyword in their description.  
1\.    When you are on the search results page, click on “More Options.”

  
 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/d4913ece-b4a1-422c-8811-e661964e500c)

  
2\. As highlighted below, search the property via “Keyword Search” and click on “Apply”. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/d9be5849-52d2-40bb-b6c9-9c4ff1450c4c)

3\. The result will show all the properties with that Keyword in their Description.

  Example: Type in "Pool" in the “Keyword Search” while you have your property Search set to   Delray Beach, FL (City),
