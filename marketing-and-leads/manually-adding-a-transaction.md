# Manually Adding a Transaction | Constellation1 Customer Hub

**Manually adding a transaction**

When the Transaction Wizard is toggled off under the System Settings tab, you may find the process of manual entry better for the way, in how you already have the information listed.

_**Please note: The State/Province will auto-populate with the same information provided in the Company. For further information, please click here:**_ [_**Company Profile Maintenance**_](https://constellation1.na3.teamsupport.com/knowledgeBase/16981163)

_**On the Transactions Landing page**_

**Click Add**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/93a24967-acc4-436d-9f7e-3bc06986c08f)

**Starting with the Property Details:**

If you have enabled automatic numbering of your transactions in the [**Advanced Settings**](https://constellation1.na3.teamsupport.com/knowledgeBase/20368643)&#x20;

the number will automatically be assigned.

* If not then you will enter the Transaction number.
* MLS if applicable
* Side represented
* Offer Acceptance date
* Close date
* Property address
* Lead Source if applicable.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/15eeb3e2-f0bf-488a-9593-21519f36eeb5)

**Next is Additional Details:**

* Property Type
* Transaction Type
* Transaction Group is specifically for Franchises
* Square Footage and List price are optional, as is Legal Description.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/a3ff993c-c9b0-40ee-bf03-7005833f2032)

**Hit Save**, that transaction will then be saved to the top of your list of transaction for entering more data, double click the address.

**Contacts card**

* Add all parties involved with this transaction can and should be entered.

**Documents**

* Any related documents can be uploaded if required.

**Commissions**

* Sale Price
* Select a Commission Rule, or manually add the percentage or flat dollar amount.
* Depending on the side represented, you will also add the Cooperating Brokerage.
* If there are any off the top deductions , they are added on the Commission Card.
* Add agent, agents, or Team.
* Edit agent if any other deductions need to be added from the agent's portion.

Be sure to hit **Save**
