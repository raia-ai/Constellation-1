# How to Add an Existing Email Activity in Action Plan Manager

To add an existing activity, choose **Select** from the email activities library. The **Select an Email Activity** overlay will display.

Enter a partial name to search for an activity OR scroll to find the activity. Click the **Select** link for the activity and click the **Add** button to add the activity to your plan. Once you are directed back to the Activity plan, select Save.
