
# Agents - Documents Tab | Constellation1 Customer Hub

This article will provide you with information regarding the Documents tab of the Agent and its use. 

Each Commissions Online account provides 5 GB of storage. If you require more, please email [commissions@constellation1.com](mailto:sales@constellation1.com) and a member of our Team will be happy to assist.  

Documents support MS Word, MS Excel, PDF, txt, etc. type files.

**Add a Document**

To add a new document to an Agent, navigate to the Documents tab and select ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/83113d02-c1ba-4530-9c17-4f0e04f37b88) at the top right.

Select Choose File and navigate to the location of the document. 

Enter a Description, toggle ON Private if you do not want this document shared with other Master Users, Management Users, Office Administrators and/or the Agent and select Save. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/24e67e06-7a76-4125-806c-8b5f487c8240)

Once the document is added, you can Edit, Delete, Mark as Private (if Private not toggled on previously) and Download.   

Edit allows you to change the Description only.  If you added the incorrect document, you will select Delete and add again. 

Download will save a copy of the document to the PC's local download folder. 

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/fadc94c6-ed9d-4da6-8f25-3b4225abbedb.png)

Please click here for information relating to the [Agent and Team Leader View of Program.](https://constellation1.na3.teamsupport.com/knowledgeBase/20920481)

If you wish to monitor your Document Storage; on the left navigation bar select **System Settings / Company**.
