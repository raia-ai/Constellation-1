
# Paradym by Constellation1 April 2021 Release Notes

Production Release #67: April 20, 2021

## RELEASE SUMMARY

  
**Flyers:** Property info now editable on the flyer template.

#### PROPERTY FLYERS

Marketing a property is important, getting the right wording on flyers is part of that marketing.  
We have updated our flyer templates, allowing you to edit and modify the text on your flyer. Giving you more control to market your listings.

Added support allowing you to edit the “Subtitle” field. Editable fields include:

*   Title
*   Subtitle
*   Description

Added the ability to enable/disable the subtitle field for any new flyers.

*   Currently, this is only enabled for the Galles Properties custom flyer.

Added the ability to enable/disable the description field for any flyer.

*   This is only disabled currently for the Galles Properties custom flyer.

Updated the preview and download feature for the Galles Flyer template; to support the new subtitle field.
