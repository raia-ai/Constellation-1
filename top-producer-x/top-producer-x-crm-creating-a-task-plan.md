# Top Producer X CRM: Creating a Task Plan

**Top Producer X CRM: Creating a Task Plan**

To start creating a task plan, first go to Marketing > Task Plan Manager.

Creating a New Plan

1.  In the Task Plan Manager, click **Add Plan**.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/addplan.jpg)
    
2.  Choose the type of plan (contact/transaction), enter a name and description, and click **Save Plan**.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/createtaskplanA.jpg)
    
3.  Click **Add Task** to begin adding tasks to the plan.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/engagedcontactsplan-2A.jpg)
    
4.  On the _Add Task_ pop-up, set the task type, details, additional information, and choose the team member to which the task should be assigned. If creating an email task, be sure to select an email template to send.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/addtaskplantaskemail.jpg)
    
    The _Task due_ field indicates the date the task will be added, relative to the plan start date or previous task.
    
    Extra details for transaction plans
    
    If creating a task for a transaction plan, you will have additional date options to choose from.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/transactiontaskdue.jpg)
    
    Once you’ve applied a transaction plan, if you change a date that a task depends on (i.e. inspection date) you will be given an option to adjust due dates for dependent tasks based on your changes.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/adjusttaskpopup.jpg)
    
    If you’re creating an email task for a transaction plan, you will have a Transaction Party Roles dropdown menu. Your selection here determines which transaction party or parties will receive the email (i.e. Primary seller, Primary buyer, All).
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/transactionpartyroles.jpg)
    
5.  Once you’ve finished setting up the task, click Save and Add Another Task to save the task and add another.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/saveandadd.jpg)
    
6.  Click Save Task when you’ve finished adding tasks to return to the task list.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/savetask.jpg)
    

Creating From a Template

1.  In the Task Plan Manager, click **Browse Templates**.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/taskplansmanager.jpg)
    
2.  Search for the desired template and click the name.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/taskplanstemplate1.jpg)
    
3.  Click **Customize Plan**.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/taskplantemplates-A.jpg)
    
4.  Edit the name and description, then click **Save Plan**.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/saveplanA.jpg)
    
    Alternate Method
    
    Click **Add Plan** on the Tasks Plans Manager.
    
    Click the _Create from template?_ toggle.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/createfromtemplate.jpg)
    
    Search for the desired template and click the name.
    
    Click **Save Plan**.
    

Editing a Task Plan  
Click on a plan in the _Task Plans Manager_ to view the task list and make changes.

![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/TaskPlan/editplandetails-2A.jpg)

Click **Edit Plan Details** to change the plan name or description. Click **Add Task** or **Add Day** to add additional tasks to the plan. Edit a task by clicking the _Edit task_ icon (pen). Delete a task by clicking the _Delete task_ icon (trashcan). Click the _Print icon_ (printer) to create a printable list of the tasks on a plan.

Duplicating a Task Plan  
Save time when creating a new task plan by copying an existing plan. To do so, click on the plan to view plan details and click **Duplicate Plan**.
