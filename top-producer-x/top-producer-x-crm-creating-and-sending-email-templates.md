# Top Producer X CRM: Creating and Sending Email Templates

**Top Producer X CRM: Creating and Sending Email Templates**

Personalize your email marketing by creating custom email templates in Top Producer® X CRM. Templates you create can be set to auto-populate customer information, saving you time and maximizing your productivity.

Creating an Email Template  
To create an email template, follow these steps.

1.  Go to **Marketing** > **Template Library** > **Email**. You will see a list of all available email templates, organized by category. Click on a template to view the contents.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/EmailTemplates/emaillibrary1.png)
    
2.  If you would like to start with a blank email template, click **Add from Scratch**.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/EmailTemplates/emaillibrary2.png)
    
    To use an existing template as a base, open the template viewer by clicking **Add from Library**. Select the template you would like to use, then click **Customize Template**. This will create a copy of the template for you to modify.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/EmailTemplates/templib4.png)
    
3.  Enter a name for your template, select a template category, enter an email subject, and the body of the email.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/EmailTemplates/templib12.png)
    
4.  To enter a merge code, such as customer name, click the **Template** button tpxtemplatebutton. Search in the template list for the merge code you’d like to add, and click to add it to the email.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/EmailTemplates/templib17.png)
    
    The merge code will be inserted into the body of the email.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/EmailTemplates/templib14.png)
    
    When you later compose an email, the merge code will be automatically replaced with the appropriate information.
    
    List of Merge Codes  
    
    |     |     |     |
    | --- | --- | --- |
    | **Name** | **Code** | **Description** |
    | First name | contact.primaryPerson.firstName | The first name of the contact. |
    | Full name | contact.fullNameFirstNameFirst | The full name of the contact. |
    | Last name | contact.primaryPerson.firstName | The last name of the contact. |
    | Spouse first name | contact.secondaryPerson.firstName | The first name of the contact’s spouse. |
    | Spouse last name | contact.secondaryPerson.firstName | The last name of the contact’s spouse. |
    | Address | contact.address | The contact’s full address. |
    | Street address | contact.address.addressLine | The contact’s street address only (i.e. 1234 Main St). |
    | Agent name | agent.agentName | Your full name. |
    | Agent first name | agent.firstName | Your first name. |
    | Agent last name | agent.lastName | Your last name. |
    | Agent email | agent.agentEmail | Your email address. |
    | Agent business phone | agent.businessPhone | Your business phone number. |
    | Agent mobile phone | agent.mobilePhone | Your mobile phone number. |
    | Agent website | agent.agentWebSite | Your website. |
    | Office address | agent.agentAddress | Your full office address. |
    | Company website | agent.companyWebsite | Your company website. |
    | Office map link | agent.googleMapsLink | A link to your office on Google Maps. |
    
      
    Note: Agent and office details can updated under **Settings** > **My Information**.
5.  Once you’re done editing your template, click **Save Template**.

Congratulations, your template is now ready for use. If you’d like to make any changes, click the template name in the _Template Library_ to open the template for editing.

Sending an Email Template  
To send an email template, follow these steps.

Note: To send email, first connect your email account. See Integrate Your Email Account for more information.

1.  Click the **Compose** button compose and select **Send Email** sendemail1, or click the Compose Email button in the Email tab of a contact record.
2.  Click the **Template** button to view merge codes and email templates. Your personal templates will appear at the top. Click the one you wish to apply.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/EmailTemplates/templib15.png)
    
3.  The template will be automatically applied to the email, and any merge codes will be auto-populated.
    
    ![](https://na3.files.teamsupport.com/TSData/WikiDocs/2472652/images/TopProducer%20-%20Wiki%20KB%20images/EmailTemplates/templib16.png)
    
    Edit the email as desired, and click **Send Email** to send.
