
# BOI - MLS Memberships and Configuration

**BOI and MLS Memberships BOI Configuration**

Attribute source of record can be configured per Listing Source at the RECo level for Agents / Offices in the table “AttributeSourceOfRecord”.

By adding entries one can set the source of record per MLS entry in the column DataSourceID \[defined in dmnDataSource table\]. 

The definitions for the source of records available to the client are held in table “dmnAttribute”. 

  
**NOTE: All MLS sources not in table dmnAttribute must be added.**
