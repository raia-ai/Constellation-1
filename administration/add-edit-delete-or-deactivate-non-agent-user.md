# Add, Edit, Delete or Deactivate Non-Agent User

This article provides the steps to add, edit, delete or deactivate a non-agent user.&#x20;

**Add a Non-Agent User**

Once logged into the application, on the left navigation bar select **System Settings / Users,** then select Add button at top right of the window.

When prompted, toggle OFF **Is Agent User?** and select Next.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17637493)

Users are welcome to complete all the User Profile fields as needed, however minimally they must complete the mandatory fields (designated by \*).&#x20;

Users will select the Permission Level they wish to allocate to this user. If user is not sure which Permission Level to assign, please review; [Viewing and Editing User Permissions](https://constellation1.na3.teamsupport.com/knowledgeBase/15339324) or ask your Office Manager/Administrator for guidance. If the Permission Level of Master User is selected, the Assigned Office will be greyed out because this Permission Level applies to all Offices.&#x20;

Role will only be displayed if the company belongs to Anywhere Real Estateâ„  (previously known as Realogy).&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/24d87c63-11ce-45a1-ab30-b7e71d97840d)

"+ Add New" will allow you to configure different Permission Levels based on Offices.&#x20;

Once all of the information is entered, select Save & Invite user which will send an email to the new user with their username and password.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17637495)

If the new non-agent user does not see the invitation in their Inbox, they should check their spam folder.   &#x20;

**Edit a Non-Agent User**

Once logged into the application, on the left navigation bar select **System Settings / Users** and click on the ellipsis to the far right of user to select Edit.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/73572bed-5338-4d7c-92c5-b0673ba0a028.png)

Make your edits and select Save.

**Delete a Non-Agent User**

Once logged into the application, on the left navigation bar select **System Settings / Users** and click on the ellipsis to the far right of user to select Delete.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/91aa4c0c-62b9-4b51-a1cf-062947bea31f.png)

You will receive the following prompt, make your selection.&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/88006c22-a468-47c6-86e3-ef84e8e6bedd)

**Deactivate a Non-Agent User**

Once logged into the application, on the left navigation bar select **System Settings / Users** and click on the ellipsis to the far right of user and select Deactivate.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/4dedde85-2029-49e2-ab71-4b3e266bc1d8.png)

You will receive the following prompt, make your selection.&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/35a5d8e6-99a0-4351-8c26-709c88112300)

Inactive users can be found by selecting the Advanced Filters icon at the top right of the Users landing page, select Inactive and Apply.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/6f3a5f5c-33d5-4111-af0c-7944aa81f73d)

If an Advanced Filter is active, the icon will change to blue. To reset the list back to Active Users, select the Advanced Filter icon, select Active and Apply.&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/15586ec3-d04f-4360-8c34-1734cd7961c5)
