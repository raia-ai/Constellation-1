
# Users Section | Constellation1 Customer Hub

*   In order to add a new user to your account, navigate to the "Users" page located under the drop-down menu next to your account name.

            ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/9d371c9a-db66-4e49-84d4-1e4a69f7e875)

*   To add a new user, simply click on the "Add" button located in the top right corner of the Users page.

                   ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/6f340fe7-9cee-469f-8f3b-c63170c0aae1)

*   Provide all the necessary information in the required fields. After that, choose a username and password.

**Note:** Always include the same username in the EmailToAddress field before @reloemail.com, so that all responses and emails are sent directly to the software.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/faeee865-5e7f-4bb9-bc72-20db2257ea79)

*   Click on the "Access" tab, choose an access level from the drop-down menu, and press "Save" to create a new user.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/9ea19422-80bd-4d4a-8707-868f765855c3)

*   Selecting an “Agent” access level will create a new user who can access and view only the referrals in which the user is selected as an Assigned or Referring agent.

**Note:** You can add as many agents as you want.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/cfcfed39-ce85-4f1e-ae5f-3186b579f764)

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/1e37c42f-258e-4c25-9693-cd81497e43cd)

*   Selecting a “Corporate Member” access level will create a new user who can access referrals based on the "referring agent office name" similar to Agent access.

**Note:** You can add as many Corporate Members as you want.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/762501f2-bdb7-427c-b91e-b4f8a2fd3ac3)

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/deb2b90e-2864-4a24-86de-5c1216eb36b7)

*   Selecting a “Team Member” access level will create a user who can access everything in the system, the only exception he is not authorized to create new users and edit any existing ones

**Note:** You will be charged an additional fee per year when you create a user with this access level.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/93e318c8-7d8b-45a2-9361-c9c628fa3171)

                ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/fc27930b-3905-43dd-8f1a-cae931f48f90) 

*   Selecting a “Relo Admin” access level will create a user who can access everything in the system, they can create new users, edit the current user information and change their access level.

**Note:** You will be charged an additional fee per year when you create a user with this access level.
