
# How to add Transaction Coordinator in a Branch for any TransactionPoint Account

**Steps to Add Transaction Coordinator in any Branch:**

1.  Login to the Admin TP Account of the specific company
2.  Click on the specific branch under the view branch list 

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/e5bc28e9-6c1d-4138-bf61-653e81015c63.png) 

3.  Click on the Staff under the View Staff list

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/48e70ca8-3cbb-4dd8-811b-b84eaa255efa.png) 

4.  At the top right corner there is an option for Add Staff click on it

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/b045bc8d-11ad-45ef-ac74-1e3536ebe96d.png) 

5.  On the Staff Registration page click on the Transaction Coordinator option at the top and provide the relevant information of the Transaction Coordinator as shown in the below screenshot and then click Finished button at the end of the page.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/ec49ace6-b055-46db-ad6c-8cdeae6221aa.png) 

This whole activity will enable you to add a new Transaction Coordinator for a specific branch and the same whole process is followed to add Executive level accounts for branches including the Admin accounts.
