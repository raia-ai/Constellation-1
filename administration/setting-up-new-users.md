
# Setting up New Users | Constellation1 Customer Hub

This article is to give step by step instructions on setting up a new user with their own username and password as well as setting their permission levels.

Navigate to the Set up Menu, Users/Passwords 

*   To add a new user, select New icon, enter User Name and then select the **Reset Password** button. Leave Current Password field blank, type the new password in the New Password and Confirm New Password field and select Reset Password button. 
*   Close the window. 
*   The **Set Program Access** button provides users the ability to control who has access to the various menu's/sub-menu's of the application. By default, all users will have access to all menu's and sub-menu's, simply remove check from applicable options to restrict access. When access is restricted for a user, the menu/sub-menu will be greyed out.
