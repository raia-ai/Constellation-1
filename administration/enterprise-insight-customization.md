# Enterprise Insight-Customization | Constellation1 Customer Hub

Once your Enterprise Insight environment is created you have the ability to add/edit users within the environment as well as customize your Groups.

Should you subscribe to White Labelling, refer to this link for further information [White Labeling in Enterprise Insights](https://constellation1.na3.teamsupport.com/knowledgeBase/23611286)

**To add a new user**

From the left navigation bar, select Users, select the Add+ button, complete the user profile with minimum First name, Last Name and email address and select:

**Save**\-in order to save the profile to send sign in credentials later or

**Save and Invite** **user**\-in order to save the profile and send sign in credentials immediately

**To edit a user profile**

From the Users landing page, to the right of the user, select the ellipsis and click on **Edit** or select the user to open their profile in edit modeÂ 

Information can be edited and saved including limiting access for the user to a Permission Level of Limited Access and selecting the specific brokerages they deal with under Brokerages

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/5f85455d-ba2b-46b7-9155-d9b55f6f58c8)

**Permission Levels**

**Master User** has access to data for all brokerages and all functions

**Limited Access** can only view data for assigned brokerages with Users and Administration menu items hidden and Group Filters disabled

**To deactivate a user profile**

From the Users landing page, to the right of the user, select the ellipsis and click on Deactivate and select **OK** when the system asks you to confirm the user deactivation

**Administration of Enterprise Insight**

From the main menu on left hand side, select Administration and you will be taken to the Groups landing page

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/012fa35d-96a7-4d4a-811a-9ab914d22629)

At this point user can create Groups based on the databases included in environment by selecting **\+ Add Group**, assigning the Group a name and selecting the appropriate Brokerage(s) from the list and selecting **Save**
