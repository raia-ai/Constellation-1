# Enlarge Icons and Text Size Display in Remote Desktop Connection – Windows 2012 Server

This article provides instructions to enlarge icons and text size via the Remote Desktop Connection.

*   Log into the remote session, right click anywhere on the screen and select “**Screen resolution**”

 ![](http://emphasysrre.wpengine.com/kb/files/2020/10/Screen-Resolution.png) 

*   Click on “**Make text and other items larger or smaller**” 

![](http://emphasysrre.wpengine.com/kb/files/2020/10/Text-Larger-1024x575.png) 

*   Select “**Let me choose one scaling level for all my displays**” 

![](http://emphasysrre.wpengine.com/kb/files/2020/10/Scaling-1024x571.png) 

*   Select “**Medium**” or “**Larger**” or “**Custom sizing options**” to change the text and icon size
*   Click “**Apply**”

 ![](http://emphasysrre.wpengine.com/kb/files/2020/10/Display-1024x577.png) 

*   Click “**Sign out now**” and re-login to the session to refresh the settings. 

![](http://emphasysrre.wpengine.com/kb/files/2020/10/Sign-Out-1024x573.png)
