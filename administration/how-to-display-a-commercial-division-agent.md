# How To Display A Commercial Division Agent

For an agent to appear on the commercial real estate page: 

[http://www.longandfoster.com/pages/commercial-real-estate](http://www.longandfoster.com/pages/commercial-real-estate) make sure the agent is associated to the VA Commercial Sales office. The internal code for this office is 4050.

[Link to the office in admin](http://longandfoster.fnistools.com/Office/OfficeUserListing.aspx?OfficeID=10258)

If the agent is associated to this office and is set to be searchable then they will appear on this page when either the user searches for them by name or by clicking the "View All" button.
