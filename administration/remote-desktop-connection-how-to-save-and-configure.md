# Remote Desktop Connection; How to Save and Configure

The purpose of this article is to provide users the steps to save their Constellation1 Remote Desktop Connection credentials, and to ensure the local printer is enabled.&#x20;

Double click on RDP file attached to the e-mail received and select Save

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/768bec37-125e-4f70-bdd2-dd19e6fa950d)

When prompted, select Desktop as the location to save the file

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/b085a91a-7417-4c65-8e26-9fc16e2262b9)

Right mouse click on RDP icon and select Edit

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/86974532-f66d-4271-87f2-30e28643f345)

Select Advanced tab, then Settings...

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KBRDC2.png)

In the Logon method field, select “Allow me to select later” from drop down and OK

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KBRDC3.png)

Select Local Resources tab, there should be a check in box to left of Printers. This will ensure that anything printed via Constellation1's remote desktop connection will be redirected to the users local printer.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/kbrdc4.png)

The Display tab allows users to modify the view and look within the remote desktop connection

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/kbrdc5.png)

On the General tab, there should be a check in box to left of “Allow me to save credentials" and Save.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/kbrdc6.png)

Select Connect and paste the password provided. Users may want to select the "view” icon to ensure there are no empty spaces before or after the password. There should be a check in box to left of “Remember me” and click OK to connect.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/920c155b-b447-4980-86bc-bcace79b1ab9)

**If the password does not save as expected, you can try to re-save the password by following the instructions detailed here;** [**How to Resave the Password for the Remote Desktop Connection**](https://constellation1.na3.teamsupport.com/knowledgeBase/25381126)
