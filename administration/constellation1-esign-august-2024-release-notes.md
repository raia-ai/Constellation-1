# Constellation1 eSign August 2024 Release Notes

Production Release: August 20, 2024

## Release Summary

**Form Field:** Added option to format the information entered into the form field. \
**Mobile View:** Updates made improved Usability.\
**Bug Fixes**

## Release Details&#x20;

Form Field Formatting\
We've updated the Form Field to allow users to apply custom formatting to limit the number of characters and the types of characters that can be entered, minimizing errors when signers enter the required information. \
For instance, zip code fields can be restricted to 5 numeric characters, excluding letters. Similarly, a document policy number field can be limited to a specific number of alphanumeric characters as needed.\
These configuration options help ensure that signers provide the correct information in the proper format, reducing mistakes during the data entry process
