
# Updating an eSign Profile | Constellation1 Customer Hub

eSign offers its users the ability to update the following information in their Profile:

*   Profile Picture
*   First Name
*   Last Name
*   Company
*   Job Title
*   Phone
*   Mobile
*   Password
*   Username/Email Address
*   Time Zone
*   Date format

  
In addition, users can also setup Email Notifications for the following events: (These can be toggled to Yes or No)

*   Session State
    
    *   Session Started (Email notification for when a Session starts)
        
    *   Session Viewed (Email notification for when a Recipient views a session)
        
    *   Session Signed (Email notification for when a Recipient signs a document)
        
    *   Session Delegate Notice (Email notification for when a Recipient delegates their signing to a third party)
        
    *   Session Declined (Email notification for when a Recipient declines a Session)
        
*   Session Invites
    
    *   Session Reminder Summary (Email notification for when a Session reminder is sent)
        
*   Session Expiration
    
    *   Session Expiring (Email notification for when a Session is nearing its expiration date)
        
    *   Session Expired (Email notification for when a Session Expires)
        
*   Session Documents
    
    *   Session Documents (Email notification when Session documents are sent to the Recipients)
        

Set Tag and Markup Settings can also be configured here, which lets users adjust the default font size for the following Signer Tags and Markups:

*   Signature
*   Initial
*   Date/Time
*   Form Field
*   Markup Text

Finally, users can set their Signatures and Initials by either drawing or typing them in, and setup their Email Signature.

**Note:** Email Signature is the default email message that gets sent to every Recipient in your eSign sessions. This can also be manually input on a per Session basis.

To access your Profile, click on your Profile icon (On the top right corner of your screen) and select Account:
