
# Accounting - How to Setup a Tax-Exempt 401K Payroll Deduction Code

The purpose of this article is to provide steps for creating a 401 K payroll deduction that is exempt from federal withholding tax.  

Note:  We are not authorized to provide advice on whether a payroll deduction is or is not exempt from federal withholding.  Consult with you CPA for guidance on what type of deductions are or are not exempt from federal withholding tax.

To create a 401 K payroll deduction code that is exempt from federal withholding tax, navigate to **Payroll > Setup > Employee Deduction Codes**. 

*   Click **Add**
*   Enter a unique **Code**, up to 8 characters long
*   Enter **Description**
*   Select **401 K** from the **Type** dropdown
*   Select **GL** **Account** from the dropdown
*   Enter **Flat Amount**, if applicable
*   Enter **Percent**, if applicable
    *   If **Percent**, is selected , select **Percent of what amount?** from the dropdown
*   Enter **Ceiling** if applicable
*   Check **Exempt from federal withholding?** box, if deduction is exempt from federal withholding tax
*   **Save**

After the employee deduction code is entered, navigate to **Payroll > Setup > Employees** and find the employee you would like to assign the deduction to.

*   Click **Edit**
*   Click on **Deductions** tab
*   Click the green plus sign (+) to add a new deduction line item
*   Select the new **Deduction** code from the dropdown
*   Enter **Flat Amount**, if applicable
*   Enter **% Amount**, if applicable
*   **Save**
