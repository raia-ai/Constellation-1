# Constellation1 Commissions September 2025 Release Notes v4.4

Production Release: September 3, 2025, v4.4

Release Summary 

This release introduces several new enhancements and fixes designed to improve efficiency, accuracy, and compliance across Commissions. Highlights include: 

*   Cleaner user profiles – inactive offices no longer display
*   Easier deposit management – added Check # column \+ editable deposit date
*   Smarter listing import – now configurable for automation
*   Compliant transaction record sheet – full deposit bookmark added to all master templates
*   More flexibility in setup – up to 12 custom fields for Agents and Transactions
*   Greater control over users – master users can add and manage custom user types and permissions
*   Improved validation – address checks now prevent errors with T4A forms

Plus, we’ve resolved issues impacting data mapping and brokerage visibility. 

Release Detail 

Inactive Office in User Profile 

Deactivated offices will no longer appear in the “Assigned Office” field of a user profile.

Transaction Detail - Deposit Table  

*   A new check number column has been added to the Deposit table
*   Deposit dates can now be edited for trust deposits, disbursements, reversals, and more.

Listing Import \- Automation Enabled 

You can now configure automated listing import to run manual, daily or weekly – saving time and reducing manual work.

Letter Template - Transaction Record Sheet 

In compliance with Real Estate Council of Ontario requirements, a full deposit amount bookmark has been added to all Transaction Record Sheet master templates.

Advanced Settings - Custom Fields 

You can now add up to 12 custom fields each in both Agent and Transaction modules. 

## Permissions - Custom User Type 

Master users now have the ability to add and manage custom user types and assign permissions as needed – giving more flexibility in defining roles. 

![A screenshot of a computerAI-generated content may be incorrect.](https://constellation1.na3.teamsupport.com/api/attachments/action/1/648dc249-eb2f-40fd-b8d6-3eefe2270b72)

## T4A - Address Validation  

Commissions now validates the company address and provides a clear error prompt if it’s missing – instead of showing an “unknown error.”

Issues Resolved 

1\. Berkshire Hathaway HomeServices Data Bridge file: Commercial transactions now map correctly to the Commercial Transaction Group 

 (previously defaulted to Residential) 

2\. Enterprise Insights: newly added brokerages now sync correctly with the license portal and display for users.
