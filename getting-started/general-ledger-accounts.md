# General Ledger Accounts | Constellation1 Customer Hub

The General Ledger Account Number, Accounting System ID and General Ledger Account Type **MUST** be the same as those configured in QuickBooks®. The General Ledger Account Name does not have to match between the two applications, but you can edit the name if you choose. 

**Setup / General Ledger Accounts**

*   Select NEW icon at top right of window to add a new general ledger account. You can also EDIT or DELETE general ledger accounts by selecting the applicable icon.
*   General Ledger Account Number - enter the number as configured in QuickBooks®
*   General Ledger Account Name - enter the account name as configured in QuickBooks®
*   Accounting System ID - enter the same number as entered for the General Ledger Account Number field
*   General Ledger Account Type - select the same account type as configured in QuickBooks®
*   You do not have to enter all of the general ledger accounts that you have set up in QuickBooks®.  You will configure the accounts used to enter/process a transactions' closing in Commissions.

#### _**Bank Accounts:**_

*   **General/Operating 1, 2 and 3 -** enter/edit the number(s) for the General/Operating bank account(s).
*   **Escrow/Trust 1, 2 and 3 -** enter/edit the number(s) for the Escrow/Trust bank account(s).
*   **Commission Trust 1, 2 and 3 (AG) -** enter/edit the number(s) for the bank account(s) that will be used to pay Sales Associates.
*   **Pay Advance From -** bank account used to issue advance check.
*   **Pay Other Brokerage From -** bank account used to pay outside brokerages.
*   **Pay Referrals From -** bank account used to pay any referrals.
*   **Pay Rebate From -** bank account used to pay any rebates to sellers/buyers.
*   **Pay Interest From -** bank account used to pay any interest earned.
*   **Pay Manager From (BR) -** bank account used to pay the Manager when calculation type = Brokerage Commission.
*   **Pay Manager From (AG) -** bank account used to pay the Manager when calculation type = Agent Commission.
*   **Pay Withholdings From -** bank account used to pay garnishments/withholdings.
*   **Pay Employee From -** bank account used when you opt to pay your Sales Associates as an employee.
*   **Term Deposit/GIC -** enter/edit the number used to track Term Deposits/GIC certificates issued and redeemed.
*   **Agent Bypass -** add the number used to capture Sales Associate commissions paid directly by the Title/Escrow Company.

#### _**Additional Accounts:**_

*   **Commission Income -** enter/edit the number used for commission income.
*   **Other Brokerage Expense -** enter/edit the number used to pay external brokerages.
*   **Escrow/Trust Payable 1, 2 and 3 -** enter/edit the number used to off-set escrow/trust deposits received from Buyer(s).
*   **Accounts Receivable -** enter/edit the number used for Sales Associates' Accounts Receivable.
*   **Agent Advance Handling Fee -** enter/edit the income number used to collect a fee charged to the Sales Associate for issuing an internally issued advance.
*   **Referral Expense -** enter/edit the number used to pay referrals.
*   **MLS/Transaction Fee Income -** enter/edit the income number used to track MLS/Transaction Fee.
*   **Brokerage Commission Income -** enter/edit the income number used to track brokerage portion of commission.
*   **Rebate Expense/Cost of Goods Sold -** enter/edit the number used to pay rebates via the referral section of a transaction.
*   **Third Party Advance Payable -** enter/edit the number used to track advances Sales Associates receive from a third party, ex. Agent’s Equity.
*   **Agent Commission Expense -** enter/edit the number used to pay Sales Associates.
*   **MLS/Transaction Fee Expense -** enter/edit the number used to track the MLS/Transaction Fee charged.
*   **Withholding 1, 2 and 3 -** enter/edit the number used to track a Sales Associates commission that is garnished/withheld.
*   **Funds Transfer -** enter/edit the number used to transfer the tax (HST/GST) portion of the brokerages split/fee, if Tax Automation is not enabled.
*   **Manager Commission Expense -** enter/edit the number used to track Manager Override/Recruiting Bonus) commission.
*   **Transaction Clearing -** enter/edit the number used to track Sales Associate advances/withholdings.
*   **Interest Income -** enter/edit the number used to track interest earned on escrow/trust deposits.
*   **Brokerage Gross Commission Income -** select the same number used for commission income
*   **Agent Gross Expense -** enter/edit the number used to track the GROSS commission paid to a Sales Associate. This account MUST be different than the general ledger allocated to "Agent Commission Expense" above in order for the 1099 in QuickBooks® to generate with the GROSS commission as you require.
*   **HST Payable -** enter/edit the number used to capture tax collected/paid on commissions.
*   **Broker City Tax Expense, City Tax Expense, Broker State Expense and State Expense -** enter the number(s) used to track Broker City, City, Broker State and State expenses.
*   **Broker City Tax Income, City Tax Income, Broker State Income and State Income -** enter the number(s) used to track Broker City, City, Broker State and State income.
*   **Unshared Income -** enter the number used to track a separate fee collected from a transactions' closing that is not subject to a commission plan. This fee will represent income to the brokerage only.
*   **HST Payable -** enter/edit the number used to capture tax collected/paid on commissions.
*   **Broker City Tax Expense, City Tax Expense, Broker State Expense and State Expense -** enter the number(s) used to track Broker City, City, Broker State and State expenses.
*   **Broker City Tax Income, City Tax Income, Broker State Income and State Income -** enter the number(s) used to track Broker City, City, Broker State and State income.
*   **Unshared Income -** enter the number used to track a separate fee collected from a transactions' closing that is not subject to a commission plan. This fee will represent income to the brokerage only.
*   **Payroll Expense -** enter the number used to track Payroll Expenses.  This should be the same account that you are using for your Employees.
*   **Enable Accounts Payable -** enter the number used for Accounts Payable if Sales Associates is paid via ACH/Direct Deposit or a third party utility Sales Associate Commission Integrate to Accounts Payable/Bills
