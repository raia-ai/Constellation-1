# Guidelines to Set Up Both the Company and the Agents for RE/MAX Franchise reporting

The configurations are typically performed during your initial set up with your trainer. If however you have recently joined as a Franchisee this is helpful information for one to be able to generate the Franchise report for your region.

RE/MAX Franchise Reports which supports an excel file upload.

* Monthly Statistics Detail Report (COR); Company Owned Region
* Monthly Statistics Summary Report (IOR); Independent Owned Region
* Monthly Production Summary Report (iFranchise); RE/MAX Integra

If you have any issues with your submissions, you must contact your RE/MAX Support Desk for assistance.&#x20;

_**Set Up & Configuration:**_

* Under Set Up and Company Profile Integration tab, ensure that Enable Franchise Integration Box has been checked.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-REMAXCompanyAgentsetup/REMAXcompanyAgentsetup1.png)

* Set Up / Branch Locations.  A branch Location needs to be set up with the following format, regardless if there is only one branch location or not.&#x20;

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-REMAXCompanyAgentsetup/REMAXcompanyAgentsetup2.png)

* On the Agent profile each Agent should have their Branch Location&#x20;

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-REMAXCompanyAgentsetup/REMAXcompanyAgentsetup3.png)

* As well as the agents Franchise ID found under Additional Details.&#x20;

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-REMAXCompanyAgentsetup/REMAXcompanyAgentsetup4.png)

_**Team Configurations**_

* Each Team must be set up with a Team Name, a Team Franchise ID and a Team Leader.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-REMAXCompanyAgentsetup/REMAXcompanyAgentsetup5.png)

**Transaction Level**

* The following fields are also required for Re/Max Reporting &#x20;
* On main Tab

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-REMAXCompanyAgentsetup/REMAXcompanyAgentsetup6.png)

* &#x20;On the Commission Tab the branch associated with the transaction
* If you already closed some transactions without the following information, there is now a new edit button for closed transactions.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-REMAXCompanyAgentsetup/REMAXcompanyAgentsetup7.png)

* The COR and ICOR Reports may also be exported to an Excel file.
* Once exported to find the file, you will receive the following message.
* Follow pathway described in the message you receive to upload to your particular region.&#x20;
