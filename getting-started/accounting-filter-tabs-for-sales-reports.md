# Accounting - Filter Tabs for Sales Reports

See below for directions on how to setup additional filtering selections for each filter tab type:  

**Projects**:  To set up a project, navigate to **System > Setup > Projects**.

*   Click **Add**
*   Enter a unique **Project Code** up to 8 alphanumeric characters
*   Enter a unique **Project Description**
*   Enter **Warning Text**, if applicable
*   Enter **Project Address**, if applicable

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/d7aad52d-e29d-4fe1-8818-103c141fa300)

Note:  If you add a **Warning Text**, to the **Project**, you will receive the following pop-up whenever you select the project, within a sale transaction:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/b99b71c0-82bf-434a-9a20-0fb290f60e82)

**Classes**:  To setup a sales class, navigate to **System > Setup > Classes**.

*   Click **Add**
*   Check the **Default for Module?** box, if applicable
*   Select **Sales** from the **Module** dropdown 
*   Enter a unique **Class Code** up to 8 alphanumeric characters
*   Enter a unique **Class Description**
*   Select **Cash Account** from dropdown
*   Select **Commission Income Account** from dropdown
*   Select **Commission Expense Account** from dropdown

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/bbabd70a-e3f5-4739-9532-22ed8265a3cb)

**Category:**  To setup a category code, navigate to **Sales > Setup > Category Codes**.

*   Click **Add**
*   Enter a unique **Category Code** up to 2 alphanumeric characters
*   Enter a unique **Description**
*   Check **Selling Side?** box, and or **Buying Side?**, box if applicable
*   Click **Save**

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/22ebb0a6-5257-4e94-92dc-2a0e7c994d5d)

**Property Types**:  To setup a property type, navigate to **Sales > Setup > Property Types**.

*   Click **Add**
*   Enter a unique **Property Type Code** up to 8 alphanumeric characters
*   Enter **Franchise Prop Type Code**, if applicable
*   Enter unique **Description**
*   Check the **Commercial?**, **Residential?** or **Vacant Land?** box, if applicable
*   Click **Save**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/f865bf18-cbe8-4e11-a6a6-1955710f1a4a)

**Commission Plans**:  To setup a commission plan, please review our article on the setup of new commission plans found at [Accounting - How to setup a New Commission Plan](https://constellation1.na3.teamsupport.com/knowledgeBase/17928403). 

**Side Represented:**  To set up a side represented code, navigate to **Sales > Setup > Side Represented Codes**. 

*   Click **Add**
*   Enter a unique **Side Represented Code** up to 2 characters
*   Enter a unique **Description**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/323517ec-7d24-4bfe-a64b-5800d9ca1556)

**Sources:**  To set up a new source, navigate to **Sales > Setup > Source of Business**. 

*   Click **Add**
*   Enter a unique **Source of Business Code** up to 4 characters
*   Enter a unique **Description**

**![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/abaf644b-e573-4dd3-9a0e-ead55cd42c53)**

**GL Codes:**  To set up a new GL code, please review our article on how to set up GL Codes found at [Accounting - How to Setup a New GL Code.](https://constellation1.na3.teamsupport.com/knowledgeBase/16894896)

**Vendors:**  To set up a new vendor, please review our article on how to set up a new vendor at [Accounting - How to add a Vendor Profile Record.](https://constellation1.na3.teamsupport.com/knowledgeBase/24924349)
