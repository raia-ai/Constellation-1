# Accounting - Import Payroll Withholding Tax Tables

The purpose of this article is to provide instructions on how to import payroll tax tables.

To import payroll tax tables, follow the steps below.

*   Download the attached zipped payroll tax table file, **TaxTable####.zip**.  Where "####" represents the payroll tax year.
*   Navigate to the downloaded file.
    *   In most instances, it should have downloaded to your **Downloads** folder.
*   Right-click on the file and select **Properties**.
*   On **General** tab check the **Unblock** box. 
*   Click **Apply**, then **OK**. 
*   Unzip the file. 
    *   In most instances, you can do this by right-clicking on the file then click **Extract All**.  
*   In the program, navigate to **Payroll > Setup > Withholding Tax Table**.
*   Click the **Import** button.
*   Navigate to the payroll tax table file and double click on it. 
*   Click **OK** to the following pop-up.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/dd1c219b-96b8-4ac4-b579-c59998f987ca)

*   Select **Non-Withholding Tax Rates** from **Payroll > Setup**.
*   Highlight applicable **Tax Year** then click **Edit**.  
*   Enter **SUTA Multiplier**, if applicable.
*   Enter **SUTA Ceiling**, if applicable.  
*   Click **Save**.
