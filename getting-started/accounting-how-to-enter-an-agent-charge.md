# Accounting - How to enter an Agent Charge

The purpose of this article is to provide the steps for entering an agent charge

To enter an agent charge, navigate to **Agents > Processing > Agent Charges**, then follow the steps below:

*   Click **Add**. 
*   Select the **Agent** from the dropdown.
    *   The agent's **Office** will autofill, based on the office selected in the agent's profile record, and is not editable.
*   Enter **Date Accrued**.
*   Enter **Amount**.
*   Select GL code from the **Account** dropdown. 
*   Select **Project** from the dropdown, if applicable.
*   Enter unique **Description**.
*   Check or uncheck **Subject to Late Fee?** box, if applicable.
    *   Note:  If you would like the system to default the **Subject to Late Fee?** box as checked, navigate to **System > Setup > System Settings** and enable system setting **APPLYLATE**.
*   Check or uncheck **Taxable?** box, if applicable. 
    *   Note:  This checkbox will only be visible if system setting **ACTAXABLE** is enabled.  If you are required to collect taxes on agent charges, navigate to **System > Setup > System Settings** and enable system setting **ACTAXABLE**.  If you would like the system to default the **Taxable?** box as checked, navigate to **System > Setup > System Settings** and enable system setting **APPLYTAXABLE**.  If you would like to apply tax to late fees, then enable system setting **ACTAXLATEF** as well.        
*   Click **Save**.
