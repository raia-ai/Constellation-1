# Reconnecting Commissions Desktop to QuickBooks® Online

This article highlights the steps required to reconnect Commissions application to [QuickBooks® Online](https://quickbooks.intuit.com/online/)&#x20;

* Reconnecting the applications is required as Intuit enhanced its security with integrated applications.
* **NOTE: The following steps should only be performed if you have previously integrated with QuickBooks® Online and you have the QuickBooks® Online ADMIN account credentials. ONLY the Admin account can re-establish the integration.**
* Launch Commissions with credentials that provide access to the Company Profile.
* Navigate to **Setup / Company Profile / Accounting** tab
* Ensure the correct QuickBooks® Online version is selected via the **Accounting System** dropdown
* Select **Verify Accounts**

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/5954f4c6-c00b-4046-855f-736f76c56cef.png)

* &#x20;You will be prompted to enter the QuickBooks® Admin login credentials;&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/7ce590c0-deb1-4e89-9e9d-92705d3a557b.png)

Select **Connect**&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/18402ef8-d2f8-487f-b57d-e9e225197922.png)

Once successfully integrated, a blue button will display **Disconnect**&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/f5df5e16-130a-4c6e-8b3a-12fbdecddce0.png)
