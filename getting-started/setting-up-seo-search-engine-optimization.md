# Setting Up SEO (Search Engine Optimization)

Search engine optimization plays an import role with any website. On the SEO page you can add a browser title. This will show as the title of your website in search engines.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/e5e5e7a3-a046-408c-bfc7-9345a93dd195)

You have the option to edit the Robot Text file. This file is managed at the system level. It is recommended to use the default setting for this.

If you make any changes to this page, be sure to click **Save** at the bottom.
