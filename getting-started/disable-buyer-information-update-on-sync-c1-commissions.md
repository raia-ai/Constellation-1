
# Disable Buyer Information update on sync (C1 Commissions)

C1 Commissions by default will ask upon the closing of a transaction if you want to update the Buyers information with the properties address that they just purchased, this only occurs when you are on the buying side of a transaction, but if you wish to not do so here are the steps to take to avoid that update notification.

*   To disable this feature, select Setup > Company Profile > System Defaults.
*   Under Additional System Settings, deselect Disable Prompt to Update Buyer's Address:
