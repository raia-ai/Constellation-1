# Setting up email functions | Constellation1 Customer Hub

Commissions has the ability to email via a transaction, letters and reports for each intended recipient.

* Navigate to Setup / Company Profile / System Defaults&#x20;
* Click “Email Document Configurations” (you do not need to be in Edit mode)&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/f0d49855-8724-439d-9a53-37bfe4a63122.png)

_**Default Email**_ _**Configurations**_ &#x20;

* In this window, you can select which Default Documents can be emailed to the various Transaction Resources. For example, the Buyer can be emailed the Buyer letter. You can also default the email body per Resource.  “MailGun” will not send an email without a body.&#x20;
* **From** **E\*\*\*\*mail** will auto-populate with “noreply@emphasysre.com” – please do not change this!&#x20;

* Select the Resource you wish to Email and the Default Documents configured above will auto display. You can opt to include more documents at this time.&#x20;
* If there is more than one Resource; use the drop down. The email address attached to Resource will auto-populate and you can change it “on the fly” if needed.  &#x20;
* If there is more than one Internal Agent attached to the Transaction, you will receive the following warning “There are multiple agents on this transaction(s). Do you wish to proceed?”, select YES or NO. Some Documents listed will include commissions for another Agent which should not be shared.&#x20;
* **E\*\*\*\*mail** **S\*\*\*\*ubject** will populate with “Documents for Transaction: Address” and can be edited on a Transaction to Transaction basis.&#x20;
* **Email Body** will populate with the body configured above per Resource OR Please see enclosed document(s) for Transaction: Address and Sent from Emphasys Software, a Constellation Real Estate Group Company.&#x20;
* Then select SEND.&#x20;
