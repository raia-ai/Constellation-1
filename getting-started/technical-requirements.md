# Technical Requirements | Constellation1 Customer Hub

**Commissions Desktop**

**On Premise Installations (Self-hosted)**

*   i5 or i7 Core™ Processor  
    256GB SSD  
    8GB RAM Memory
*   Windows 8 Professional or higher  
    QuickBooks® Desktop: Pro / Premier / Enterprise 2012 or higher  
    QuickBooks® Online: Essentials or Plus  
    Microsoft Word 2016 or higher (if opting to use integrated Letter templates)
*   Microsoft Windows Server 2012 or higher (Windows 2016 Server recommended)  
    SQL Server 2014 Express or above

**Cloud Services (remote desktop client to connect from own PC to server)**

*   QuickBooks® Desktop: Pro / Premier / Enterprise: version must be updated within last 3 years
*   High Speed Internet 
*   Microsoft Word 2016 or higher (if opting to use integrated Letter templates)
