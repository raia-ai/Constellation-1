# Accounting - System Settings For Agent Charges

Constellation1 Accounting has many ways to adjust how the Agent Charges function in regards to taxation and fees. Within **System > Setup > System Settings**, there is a series of settings that can be toggled or adjusted to alter how Agent Charges can be processed. A listing of these settings and how they affect Agent Charges can be found below:

### ACAPDESC

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/14bab3df-8125-4a8b-b0f6-d3f3ada75417)

When this setting is enabled, any bill created with an Agent Charge Entry (in Accounts Payable > Processing > Bills) will populate the Description field on the Agent Charge with the Description found in the bill:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/d3c3de3a-5c0b-42ec-9b08-d8c507783659)![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/40bf72ea-b3a7-43ae-bd62-1cb48476cda5)

### ACGLDESC

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/5bdb5542-5e3c-402f-aaa3-9ab9c6e920ee)

When enabled, this setting will automatically enter the G/L Description into the Description field of the Agent Charge as you are creating it:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/4e2a5ec8-0316-4933-be8e-7d3cb26235bb)

### ACTAXABLE

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/f4b7c9a6-e0f7-4952-adf4-2db6db37b5c8)

When enabled, this setting will populate the **Taxable?** check box on Agent Charges:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/6050de1e-30c3-4a4e-b199-80252e19dc83)

### ACTAXLATEF

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/d5bd2773-8620-4a8f-ab85-a19f7048643c)

When enabled, any late fees associated with agent charges will also be taxable alongside the charge itself if the **Taxable?** box is checked.

### APPLYLATE

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/622fe68d-751b-4425-b455-719efe5a120c)

When enabled, this setting will automatically check the **Subject to Late Fee?** option on agent charges:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/51aa374d-5be4-4057-964f-edaa2af0a902)

### APPLYTAXABLE

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/744bca04-a30a-41a8-ae0b-e7a880c73997)

When enabled, this setting will automatically check the **Taxable?** option on agent charges.

### AR\_COMMENT

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/64c44456-48fd-4c0d-85e5-1cf07ba3406c)

This setting has a text field that will allow customization of the message seen at the bottom of reports generated from **Agents > Reports > Charge Statements**:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/77f0fd1d-38fb-4a6f-8683-f801756d82d9)

Please note that this field has a maximum length of 64 characters.
