# Accounting - System Setting COMMPLANEDIT

The purpose of this article is to provide information regarding the use of system setting **COMMPLANEDIT**.

System setting **COMMPLANEDIT**, when enabled, will allow you to select a different commission plan from the **Commission Plan** dropdown in the agent's commission record of sale transactions. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/ad66ec74-0b5d-469d-b129-129f8d0c522e)

If you would like to prevent users from changing the agent's commission plan, from the agent's commission record, then you can disable system setting **COMMPLANEDT**, by navigating to **System** \> **Setup** \> **System Settings_._**

*   Highlight system setting **COMMPLANEDIT**
*   Click **Edit**
*   Uncheck **Enabled?** box
*   **Save**

**![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/8e98ac65-9ee5-456b-ab80-96880713795d)**

Note:  If you have the **Sale Transactions** tab open, while editing system setting **COMMPLANEDIT**, you will need to close the **Sale Transactions** tab and reopen it before you will see the change take effect.
