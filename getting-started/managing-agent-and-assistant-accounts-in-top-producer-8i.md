# Managing Agent and Assistant accounts in Top Producer 8i

While Top Producer Assistant accounts are no longer offered, customers with older accounts may still be using these licenses. Managing Top Producer Assistant accounts can only be done in Top Producer 8i, and the following steps will show you how.

  
1\. Log into Top Producer 8i that has a Top Producer Assistant license as part of the subscription.  
2\. Hover over the Agent’s name in the top right corner and click My Account.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/08cf564a-4f58-433f-8728-e60c6d0b3b7a)

  
3\. Click the Team Members tab in the top left corner.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/ed36a8d4-498f-4f8b-9ad3-b33f722acb52)

4\. Click the Assistants button.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/67a2735b-0d88-49f5-8ca6-9f4107ed8e89)

5\. Click on the name of the Assistant you want to edit to make changes to the account, or click Add Assistant in the bottom left if there’s an available Top Producer Assistant license that needs to be setup.  
6\. The following screen will appear, which will either be filled in if an existing Top Producer Assistant name was clicked on, or empty if Add Assistant was selected.

Here is what an Add Assistant screen looks like:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/4b36fb2f-f9ac-4a5b-af18-73cc52001dcf)

7\. Click the Save Assistant Info button to save any changes made to the Top Producer Assistant account.  
8\. Please be aware of the following:   
a) You will not be able to edit/change the username for an existing Top Producer Assistant (passwords can be changed).  
b) The “Give the assistant access to create an email mailbox” and “I want this assistant to be able to integration with Office 365, Google and Outlook” features are no longer available.
