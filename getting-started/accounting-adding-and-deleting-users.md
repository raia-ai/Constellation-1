# Accounting - Adding and Deleting Users

The purpose of this article details the steps to add a new user.

To add a new user, navigate to **System > Setup > Users**.

*   Select **Add**.
*   Enter **First Name** and **Last Name**.
*   Enter **Username**.
    *    We suggest first initial followed by last name.
*   Enter **Password** and **Confirm Password**.
*   In the **User Roles** section, select at least one user role.
*   Select the user's Check Responsibility Default from the dropdown. 
    *   Note: If you would like to setup unique check responsibility codes, navigate to **System > Setup > Check Responsibilities**. 

In the example below, we have setup a user with **Admin** permissions to **All Modules**.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/7a0c6e20-7bbf-441f-926d-121d1d976b04)

The new user can change their password, once they log into the program, by navigating to **File > Change Password**, in the upper left-hand corner of the screen. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/ba737d4d-5bb7-45ad-8d4e-6e4920e46a95)

When clicking on **File > Change Password**, the following pops up:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/5fb45ffb-e52c-4d78-a261-9680e2d7e411)

*   Enter **Current Password**
*   Enter **New Password** and **Confirm New Password**
*   Click **OK**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/b7e12c8c-d0e2-4361-ace4-1e1abe052d51)

If you would like to delete a user, navigate to **System > Setup > Users**.

*   Highlight the username in the left pane
*   Click the **Delete** button

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/dbd64562-9b26-40e5-b9d9-f8669ace8289)
