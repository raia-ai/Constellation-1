# Resolving The “Oops an Error Occurred” Prompt When Viewing a New Tour’s Analytics

If you encounter an "Oops! An Error Occurred" message while accessing the analytics for a new Tour for the first time, there's a straightforward solution to fix it.

*   Click on your name at the top right corner and select "Settings".
    *   ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/70ae4cc1-f5a4-46af-98a5-33363282d7bb)
*   On the "Settings" page, find and click on "Branding".
    *   ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/7d215a48-023e-4f02-ac38-9d7dd5d45220)
*   In the top right corner, under your name, click on "Save".

This should resolve the error on the analytics screen for the specific Tour the next time you access it. This issue is most commonly seen with newer accounts, especially during the first Tour of a listing via Paradym.
