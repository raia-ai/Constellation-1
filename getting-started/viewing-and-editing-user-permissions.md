
# Viewing and Editing User Permissions

This article outlines how to view and edit existing permissions in Commissions Online allowing users to assign different permissions to different types of users to suit your business requirements. 

Agent and Team Leader access can also be edited. Click here for further information; [Agent and Team Leader View of Program](https://constellation1.na3.teamsupport.com/knowledgeBase/20920481)

On the left navigation bar select **System Settings / Permissions**; where you will see the following tabs:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/8f7d1caa-b97a-4cff-bd68-c89952cb18b8)

*   Admin View for office/staff members
*   Team Leader View where permissions can be customized for Team Leaders 
*   Agent View where permissions can be customized for Agents

Admin View provides the following types of permissions:

*   Master User 
*   Management User
*   Office Administrator

To customize the Admin View permissions, select Admin View tab

Adding access for Management User and/or Office Administrator

*   **Please note**: Permissions can be customized for Management User and Office Administrator only. A Master User has full permissions to everything in the application. 
*   In the System & Users section is the full list of all the permissions, simply drag and drop the permission option to the applicable user setup
*   If you add a permission that already exists in error, you will be prompted with a warning to advise that the permission already exists.

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/41709b80-f5c8-4a95-8235-d59a3e08da33)

Removing access for Management User and/or Office Administrator

*   **Please note**: Permissions can be customized for Management User and Office Administrator only, a Master User has full permissions to everything in the application. 
*   Under the applicable user, simply click on the **red x** to the right of permission label ex. Removing Dashboard Access for Office Administrator User in screen shot below.
