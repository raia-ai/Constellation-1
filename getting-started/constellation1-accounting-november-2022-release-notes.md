# Constellation1 Accounting - November 2022 Release Notes

**Sales**

*   Sales>>Processing>>Issue Trust Checks:  Selection grid now refreshes after **Finish Processing** button is selected. 

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/ae1d89cb-624a-4d0f-85a7-a8c10ac6fa49.png)

*   Sales>>Processing>>Sale Transactions:  In the left pane, **Closed** and **Paid** date fields default to display the full date.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/6a0595aa-6538-4004-8cc9-5d02887ba3ba.png)

**Agents**

*   Agents>>Setup>>Agents: **Copy** button is now disabled.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/81d950f5-8670-4234-a437-3526a047d8f7.png)

**Accounts Payable**

*   Accounts Payable>>Processing>>Bills:  **Delete** button is now enabled for $0 bills.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/5dda0f58-26d0-4944-b024-cbbe54f332f9.png)

**Bank**

*   Bank>>Processing>>Non-Commission Deposits:  **Credit Distribution Entries** can now be edited, regardless of whether they have been reconciled and/or retrieved into a batch deposit, for unposted non-commission deposits only.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/49f8a05f-826d-464c-a520-62d1b33761b5.png)

*   Bank>>Processing>>Bank Reconciliation:  The **Description** column now references the Trust Record #.
    *   Note: This change is not retroactive, it only applies going forward.  Activity that was already retrieved into the bank reconciliation, with the old format, will still have the check number in the **Description** column, however, uncleared activity can be removed from the bank reconciliation and re-retrieved to display the new format.  

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/e639c508-e50f-448c-bb3a-d00ea94ec026.png)

*   Bank>>Processing>>Bank Reconciliation:  **Type** column now reflects whether the record is a **Debit** or a **Credit**.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/5455a131-3778-4be1-8cb0-4748ca7afc45.png)

**General Ledger**

*   **Projects** menu item now in **General Ledger** module, moved from **System** module.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/0e21ad04-ef34-4bf8-8220-9dea782bc9e7.png)

*   GL>>Reports:  GL account description will now wrap on financial reports, to display full description. 

**Payroll**

*   Payroll>>Reports>>Form 941:  _Unofficial_ **Form 941** added to aid in filling out official 941 form.

 ![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/9fd5f6d8-ad2d-4d83-b1dd-cb11a35fd199.png)

*   Payroll Calculations:  Calculation for Medicare deduction now considers additional Medicare tax withholding rates, for wages that exceed threshold amount, currently threshold is $200,000.

*   Payroll>>Setup>>Non-Withholding Tax Rates: Removed **Medicare Ceiling** field.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/8ec2d843-b322-411d-86dc-ff589082aae9.png)

*   Payroll>>Setup>>Non-Withholding Tax Rates: Added fields **Additional Medicare Threshold** and **Additional Medicare Multiplier**.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/b8765df0-38f0-404c-87b6-c9b853ea0ef6.png)

*   Payroll>>Processing>>Payroll:  Information **“i”** button is now displayed to the right of the Medicare deduction, if any additional Medicare was withheld. When hovering over the “i” the details of how the Medicare deduction was calculated is displayed.  

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/e042a6d4-640a-4acb-97ef-726cad0fe70c.png)

*   Payroll>>Reports:  **Payroll Edit Sheet** removed, replaced with **Unpaid** option on **Payroll Journal**.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/11029850-445b-41ad-8d50-f18f57b1fa29.png)

*   Payroll>>Reports>>Payroll Journal:  New option **Paid** and **Unpaid** added.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/3d477ada-557c-4405-bc1b-27a2a8d17376.png)

*   Payroll>>Process>>Generate Payroll: **Period Beginning** and **Period Ending** dates now default to blank.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/a7806fe5-9853-4b43-9a2d-54de115544a0.png)

*   Payroll>>Setup>>Employees: **Payroll Frequency** selection list is now alphabetized.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/28ffe290-f12c-48f1-87d2-f13cddcf3e27.png)

**System**

*   System>>Setup>>Company Information:  Added **Incorporated** check box, needed for unofficial **Form 941** report, in the **Payroll** module.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/723f0286-b72e-4638-ba52-1c43ae14331c.png)

*   Question mark **"?"**, in upper-right-hand corner, has a direct link to the Constellation1 Accounting Knowledge Base.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/21b492e2-e8c2-46bc-a908-d042fa88ef0f.png)

*   Projects menu item moved to General Ledger.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/9bdf29e9-41f9-4206-bd51-b1b1b18d255e.png)
