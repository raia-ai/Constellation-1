# Integrating QuickBooks® Online with Commissions Online

This article provides the steps to integrate Commissions Online with QuickBooks® Online.

_**NOTE: The following steps should only be performed if you have the QuickBooks® Online ADMIN account credentials. ONLY the Admin account can establish the integration between the two applications.**_

Once logged into Commissions Online as a Master User or any user who has permissions to Integrations menu:

*   On the left navigation bar select **System Settings / Integrations** 
*   Select the QuickBooks®  icon

  ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17770341)

*   Select **Online**, then **Next**

 **![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/c72ff0a9-8fd1-4e56-a0f5-d317afebf725)**

*   You will be prompted to enter the QuickBooks® Admin login credentials; select **Sign In**

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/20556d89-24bb-4f7c-8693-f27039ba4972)

*   Select your company from the drop down, then **Next** 

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17770344)

Once completed and you receive the confirmation window below, select **Close**

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/d821296f-eeac-449e-b0b4-e20cf0abb674)
