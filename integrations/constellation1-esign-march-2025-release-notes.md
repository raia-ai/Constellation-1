
# Constellation1 eSign March 2025 Release Notes

Production Release: Tuesday March 25, 2025

## **Release Summary**

*   Small UI improvements.
*   Including the signer decline reason as part of the Get Session API call.

**Release Details**  
Signer Decline Reasons  
Now, the signers decline reasons are returned as part of the Get Session API call as it was requested by customers.

Bug Fixes

1.  GPES-5256  
    Mobile enhancement to the “Start KBA” button, so it’s fully visible on smaller devices.
2.  GPES-5225  
    Prevent the user from manually entering past session expiration date.
3.  GPES-5134  
    Synching the session history’s order between the session info page and the COA.
4.  GPES- 2974  
    Corrected the country selection when adding recipients.
