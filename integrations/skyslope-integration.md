
# Skyslope Integration | Constellation1 Customer Hub

This article will outline how to configure Commissions for the MLS/TMS import from Skyslope as well as the fields imported.

## Configuration

*   Go to the Company Profile / Integration Tab / MLS/TMS Integration panel. You will select and/or add/delete the Status of the transaction to Import. 

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17869775) 

*   Once you have made your selection(s), save the Company Profile.
*   You will close and relaunch Commissions in order for the configuration to take effect.

## Importing from Skyslope  

In Commissions, select MLS/TMS Sync on the left sidebar.

The Modified Since will display the date that you last imported and import any transactions that have been changed since that date.

Select Sync and you will be presented with the list of transactions that can be imported.

You can select one transaction, multiple transactions or all transactions by placing check in applicable box.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17869776) 

Select Import button and when the import is completed, you will be presented with a Transaction Data Integration Result window.

Partial Success means that all the data fields that we can import, were imported successfully. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17869777)

**_Please Note: Commissions will import the agent attached to the transactions as long as their email in SkySlope matches the email in Commissions and SkySlope “sync’s” to the Emphasys Hub every 3 hours as follows:_**

Below is a list of the fields that are imported for both Listings and Transactions.

Listings:

*   MLS number
*   Full Address (Street Number, Street Direction, Street Name, City, State and Zip Code) List Price (Selling/Leased Price)

Transactions:

*   Transaction Side (side represented; selling, buying or both)
*   Internal Agent
*   MLS number
*   PID number
*   Acceptance Date (Effective Date)
*   Closing Date
*   Full Address (Street Number, Street Direction, Street Name, City, State and Zip Code) Sale Price (Selling/Leased Price)
*   Seller
*   Buyer
*   Title/Escrow Company
*   Office Gross Commission
*   Referral

**_Please note that there is a character limitation for buyers or sellers First name or Last name which cannot exceed 25 characters, and Company Names cannot exceed 41 characters._**
