# How To Add Facebook Pixel Code To Agent Website

## ADDING THE CODE TO THE AGENT WEBSITE

1.  Login to the Agent's account
2.  On the left hand navigation, click on "My Primary Website"
3.  Click "Analytics" 
4.  Copy the Facebook Pixel code
5.  Paste the code into the "Tracking Code" text field ![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/b27cfcfe-dbcb-4ac7-a877-225f75c7c75f.png)
6.  If Google analytics is being used, you can add the Facebook Pixel code below or above the existing code **(skip if not applicable)**. 
7.  Submit

**Note:** In order to add the Facebook pixel code, ensure the agent has access to the enhanced user packages to unlock the analytics page. The standard package does not include analytics.
