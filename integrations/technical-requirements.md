# Technical Requirements | Constellation1 Customer Hub

**Commissions Online**

*   Any modern browser
*   High-speed Internet
*   Microsoft Word 2016 or higher (if opting to use integrated letter templates)

If using Integration with Commissions Online:

*   QuickBooks® Online: Essentials, Plus or Advanced
*   QuickBooks® Desktop: Pro / Premier 2014 or higher and Enterprise 14.0 or higher
*   Xero®
*   .NET Framework must be installed on the same PC if using QuickBooks® Desktop
