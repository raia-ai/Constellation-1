# How to Create the Backup of QOC from SQL

Quick Office Commander (QOC) is exclusively available to Broker Offices. This powerful tool empowers Brokers and Managers to efficiently support their Sales Representatives, ultimately enhancing the service provided to Buyers and Sellers. This creates a win-win situation for everyone involved.

The article will walk you through creating a backup of QOC from SQL.

• Select Start > Programs > Microsoft SQL Server 2005 or 2008 R2> SQL Server Management Studio Express  
• Connect to Server window should show the server name and the instance that was created for Commander Back Office, ex: server\\QUICKOFFICE.  
• Click on Connect.  
• This will open SQL Management Studio Express.  
• Once you are connected, click on the Database folder (Top left corner).  
• Right-click on QOCEE and Tasks > Backup.  
• Select Add and select File Name.  
• Click the \[…\] button, select a folder to save the backup, and provide a file name and extension ".BAK" for the SQL backup file.  
• Click OK to close the window and select OK to begin backup.

You should receive confirmation that the backup was successful, click OK and close SQL Server Management Studio Express.
