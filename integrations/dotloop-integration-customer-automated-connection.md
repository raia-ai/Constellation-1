# Dotloop Integration-Customer Automated Connection | Constellation1 Customer Hub

The purpose of this article is to provide the steps to configure Dotloop Integration within Constellation1 Commissions.

From the left navigation bar select **System Settings / Integrations** and on the Transaction Management card select **Dotloop**Â 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/1832e902-b67f-4c5e-b71f-aa5276253991)

You will be presented with the window below, select **Connect**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/9f557b34-70e7-479f-8447-ec9b1afe3762)

When redirected to the Dotloop login screen, enter your credentials and select **Continue**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/a01a9dff-5a34-4e1f-b2e7-8077a34c3dc9)

When prompted with the window below, select **Approve**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/3ad809d5-30d2-42bc-b4c8-a9b6c9b45cdd)

You will receive the following window confirming success, select **Finish**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/7a08b72d-3baf-4271-a041-2689a888643a)

For further information with the step by step process, please click here; [Dotloop Integration](https://constellation1.na3.teamsupport.com/knowledgeBase/19166627)
