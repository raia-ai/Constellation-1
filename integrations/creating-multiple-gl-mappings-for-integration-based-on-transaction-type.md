# Creating Multiple GL Mappings for Integration Based on Transaction Type

This article details how to establish multiple General Ledger mappings based on Transaction type.

Occasionally offices will want to have certain transaction types posted to different general ledger accounts in QuickBooks® than others.  Constellation1 Commissions Online allows you to copy and edit your existing GL mappings and assign to specific transaction types

To create a copy of existing General Ledger mappings to customize

*   Navigate to System Settings / General Ledger
*   Click on the Main tab and select Copy

The system will create a copy of the Main general ledger mapping settings similar to below.

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/fac036d2-43c9-4255-a25c-c59a454358d8)

Click on the generic name, in this case “mapping7” and rename to a name of your choosing such as **Lease** as noted below and in the Transaction Type dialog box, select the specific transaction type(s) you wish to use these settings for similar to below:  

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/78b4c44a-dbdb-4d24-bf25-82550f4b61ec)At this point, you can now modify the specific QuickBooks® general ledger accounts that you wish to use for this type of transaction.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/4a8fc35e-1b94-4702-bf69-71a7d0738154)

Select **Save**

Any time you close a transaction that matches the transaction type assigned to your new mapping, similar to below it will process according to these new GL accounts

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/4d1dc4be-5f9f-4649-8166-edf70a914ba2)

Should you wish to use a customized set of mappings that does **NOT** match the transaction type, you can navigate to the Commissions tab within the transaction and from General Ledger Profile dropdown, you can select which mapping profile you desire on that transaction

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/8001e748-6969-4e75-86cb-a3561fa0e82c)
