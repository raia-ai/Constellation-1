
# QuickBooks® Online Vendor, Customer and Employee Agent Name Requirements for Commissions Online Integration

This article details the required changes to your Agents names in the Vendor, Customer and Employee lists in QuickBooks® in preparation for Commissions Online integration. 

_**Note: You do not need to change any other Vendor, Customer or Employee names, only the names that are associated to your Agents.**_

1\. Vendors – the naming schema required for the integration will be “Lastname, Firstname”.  If the agent transacts as a company, the Vendor name will be their company name, for example Lacey Blackman Real Estate, LLC

The name will be entered in either the First name and Last name OR Company fields.

The \*Display name as will be Blackman, Lacey and there should be no check to left of Use display name.

The name should display as Lacey Blackman for the correct check/cheque name to use.

2\. Customers – the naming schema required for the integration will be “Lastname, Firstname.” (there is a dot/period at the end). If the agent transacts as a company, the Customer name will be their company name, for example Lacey Blackman Real Estate, LLC. (there is a dot/period at the end) 

The name will be entered in either the First name and Last name OR Company fields.

The \*Display name as will be Blackman, Lacey. (there is a dot/period at the end) and there should be no check to left of Use display name.

The name should display as Lacey Blackman. (there is a dot/period at the end) for the correct check/cheque name to use.
