# 1/6/2025 - Upcoming System Conversion for New Jersey MLS Data Feed

We want to inform you of an important update that may affect your use of the New Jersey MLS data feed.

**Key Changes:**

1.  **OSN Update:**  
    Effective immediately, the OSN for this data feed will change from **NewJerseyMLS** to **NewJerseyIDX**. Please update your systems accordingly.
    
2.  **Vendor System Conversion:**  
    On **Monday, January 6th, 2025**, a vendor system conversion will be implemented for this data feed. To help you prepare, updated data is now available for testing. You can access it by adding the `$reconvert=true` flag to your queries, along with the new OSN (**NewJerseyIDX**).
    
3.  **Key Value Changes:**  
    As part of this conversion, key values across all resources—such as `ListingKey`, `MemberKey`, and `OfficeKey`—will be updated. Please review and adjust your integrations to accommodate these changes.
