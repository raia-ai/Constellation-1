
# Disbursing Trust/Escrow Deposits | Constellation1 Customer Hub

This article highlights the steps to Disburse a Trust/Escrow Deposit in Commissions Online.

Navigate to the Details Tab of the transaction and scroll down to the Deposits card.  Ensure the "Held By" field has your company selected. Select **Action** button, and then **Add Deposit / Disbursement**.  

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/0cf5ffa8-9a8f-4914-8661-5e5ee9567cfa)

*   Select Type = Disbursement
*   Enter the applicable date, the amount will automatically populate with the full amount of deposit being held which can be overwritten as needed, select the applicable Payee from the drop down and enter comments as needed. 
*   If user tries to disburse more money that the full deposit, they will receive the following warning. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/e5e6c2e8-4f55-4f98-baf5-f5440855698a)

*   If user wants to generate a check/cheque immediately, ensure **Sync to Accounting System** toggle is ON, and select **Save & Post**.
*   If user does not want to integrate a check/cheque immediately, leave the **Sync to Accounting System toggle OFF**, and select **Save**. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/f9d9af52-bd2a-4328-80f1-50469bbe44f6)

If user does not integrate at all, the Sync to Accounting System toggle will be disabled and user will have options to **Cancel** or **Save only**.  

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/f52dd201-4f87-4e02-8e22-4c1fd0f43be1)

If user did want to integrate the check/cheque and they selected **Save** instead of **Save & Post**, on the left navigation bar select **Accounting /** **Trust Deposits** tab.

*   Select the appropriate bank account if there is more than one
*   Place check to left of the disbursement(s) user wants to integrate and select Chain link icon at the bottom right of the window. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/d1242e3e-55fc-4562-a8e5-5d30bab72f0b)

If an incorrect disbursement date was entered, it can be edited in Commissions Online by selecting **Edit** via the ellipsis in the **Actions** column to the far right of the Disbursement entry.

****_Please note, this will only change the date in Commissions Online. If changes are required to your Accounting application, you will need to manually edit that date too._****
