# Adjusting Screen Resolution for RDC (Remote Desktop Connection) on Mac and Windows Operating Systems

The purpose of this article is to provide instructions on how to adjust the screen resolution for your RDC (Remote Desktop Connection).&#x20;

Follow the instructions below to adjust the resolution for your RDC (Remote Desktop Connection) on your Windows or Mac computer.  Note: Required screen resolutions are typically **1920 x 1080** or higher but can be lower for some applications.&#x20;

**Windows OS**

* Right-click on RDC (Remote Desktop Connection) icon and click **Edit** option.
* Click on the **Display** tab.
* Adjust display setting, to obtain **1920 x 1080** or higher.
* Navigate to **General** tab.
* Click **Save**.
* Click **Connect**.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/bc25cae4-404b-4c1d-90a3-a671d3823b84)

**Mac OS**

* Open Microsoft Remote Desktop app
* Hover the mouse over the RDC connection and click the "pencil" icon to edit.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/869c7856-6e74-4c9a-8c9d-d82f2d915495)

* Adjust the **Resolution** settings.
  * Select **1920 x 1080** to obtain the desired screen size.
    * Note: You may need to adjust the screen resolution further, to get the desired effect due to your monitors native screen resolution.
  * Check **Use all monitors** box, if applicable.
  * Check **Start session is full screen** box, if applicable.
  * Check **Fit session to window** box, if applicable.
* Adjust **Color quality** settings, if applicable.
  * Check **Optimize for Retina displays** box, if applicable.
  * Check **Update the session resolution on resize** box, if applicable.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/f4cc5364-fec9-4b16-b736-6e1c2cb1e791)

* **Save** the settings and try again.
