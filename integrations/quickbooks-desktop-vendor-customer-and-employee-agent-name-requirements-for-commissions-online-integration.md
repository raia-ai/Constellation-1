
# QuickBooks® Desktop Vendor, Customer and Employee Agent Name Requirements for Commissions Online Integration

This article details the required changes to your Agents names in the Vendor, Customer and Employee lists in QuickBooks® in preparation for Commissions Online integration. 

_**Note: You do not need to change any other Vendor, Customer or Employee names, only the names that are associated to your Agents.**_

1\. Vendors – the naming schema required for the integration will be “Lastname, Firstname”.  If the agent transacts as a company, the Vendor name will be their company name, for example Lacey Blackman Real Estate, LLC

The name should be entered in either the Company Name OR Full Name fields.

   ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/743346ec-1312-4b85-a3ee-cbe0827b8ab2)  ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/40f345aa-4748-4bb4-8afe-47007c29d32d)

On the Payments Settings tab, please ensure the correct name is entered in the Print Name on Check/Cheque As (Individual or Company).

  ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/055078a2-bd9d-474c-980a-3af3680d5369)

2\. Customers – the naming schema required for the integration will be “Lastname, Firstname.” (there is a dot/period at the end). If the agent transacts as a company, the Customer name will be their company name, for example Lacey Blackman Real Estate, LLC. (there is a dot/period at the end) 

The agents name should also be entered in either the Company Name OR Full Name fields.

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/9559e9c3-2373-4fa6-aacc-53b247a7b9c4)

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/4b2e5750-2391-48ee-8db1-c1310ed7a836)

3\. Employees – the naming schema required for the integration will be “Firstname Lastname {Agent}” and should display comparable to screen shot below.  {Agent} will ensure the name is unique and NOT display on the payroll check/cheque.
