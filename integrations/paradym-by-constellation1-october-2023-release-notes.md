# Paradym by Constellation1 October 2023 Release Notes

Production Release October 2023

## Release Summary

**New Data Source for Listings:** Updated listing data source.  
**Auto Tours:** Added garage data to properties when Auto Tours are created. 

## Release Details 

Updated Listing Data Source  
We’ve updated the source for the listing data that appears in Paradym. The new source provides more listing information and a more stable environment.  
With this update, users must acknowledge and accept a new listing data agreement. Users will see an informational banner on their Paradym dashboard to prompt them to contact Paradym by clicking the Contact Us button to accept the new agreement.

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-ReleaseNotes/Paradym/2023/10%20Oct/Paradym-Oct-SyncData.png)

Garage Data for Auto Tours  
We’ve added Garage data to Auto Tours. When Auto Tours are created, the garage data will now show the garage information for the property.
