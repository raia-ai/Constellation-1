# Anywhere Real Estate℠ (formerly Realogy) Integration > Configuration

The purpose of this article is to ensure your application is configured correctly for Anywhere Real Estate℠ (formerly Realogy) Integration.

**Company** 

On the left navigation bar select **System Settings / Company**; ensure the Franchise Designation displays the correct Anywhere Real Estate℠ franchise (Century 21, Coldwell Banker and ERA ONLY). The Integration Key will be provided by a member of the Customer Experience/Development Team. Select Save.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/c1b18825-2375-44c8-b884-d19884bdd4ae)

**Offices**

On the left navigation bar select **Offices.** Ensure the Office Franchise ID is entered for each office as needed and Save.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/1880ebc6-9773-40de-89fd-a4dc27980381)

**Transaction Type, Property Type and Lead Source** 

On the left navigation bar select **Advanced Settings** / Field Settings and add, edit or delete to comply with Anywhere Real Estate℠ requirements and select Save.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/2e4265c5-03d7-41c9-bbb4-8abfcc91a52b)

**Agents**

Agents Franchise ID will reverse sync from Anywhere Real Estate℠ so there is no Franchise ID data field attached to Agents. 

On the Additional Details card of the Details tab of Agent, you must search for and select the applicable MLS Board. This is mandatory. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/14c62107-0407-414b-87e0-295dd4e449a9)

**Teams**

From the home page, select Agents / Teams tab to ensure each team has a Team Franchise ID entered and select Save.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/6f49cba8-7e9f-4b9e-bfeb-74e78967f01b)

**Keyperson**

In Commissions Online, a keyperson can be defined as all users except an Agent; for example the Master, Office Administrator or Management user.

**Transactions**

1\. On the Details tab, expand the Franchise Details card to Exclude From Reporting as needed. When toggled ON, this transaction will not be included in your franchise report. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/8345cdb1-3588-41eb-946f-540d9fad35ea)

2\. On the Commission tab of the transaction, you can ensure that any referrals paid are not included.  To do this, select the ellipsis (...) and Advanced Options.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/1915af95-6447-4e89-b429-2092290ffb78)

On the Referral Advanced Options window, ensure you select the appropriate Referring Franchise and toggle on "Exempt from Franchise Fee" and select Save.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/a5e2f4d5-acf0-453d-905e-680751a1fe16)
