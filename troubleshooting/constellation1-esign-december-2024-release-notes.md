# Constellation1 eSign December 2024 Release Notes

Production Release: December 17, 2024

## Release Summary

**Mobile View Usability:** Updated the zoom function on documents when creating a signing session.\
**Bug Fixes**

## Release Details

Session Creation Mobile View Document Zoom \
The eSign mobile view user experience now allows signing session creators to zoom in and out on individual documents when creating a signing session.&#x20;

This targeted zoom functionality improves usability by enabling the session creators to focus on specific content and precise placement of tags and markups within a document.

This update is specific to users of Android devices. The update for iOS is coming soon. &#x20;

Bug Fixes

1\. GPES-5068\
We fixed an issue with the documents being renamed when uploaded to a template. \
Previously, when uploading a document to a signing session created from a template, the document was being renamed.\
Now, this issue has been resolved.&#x20;
