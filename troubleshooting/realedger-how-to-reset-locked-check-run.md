
# REALedger - How to Reset Locked Check Run

The purpose of this article is to show how to reset a locked check run.

If you try to issue checks and receive the following pop-up, it means that you have a locked check run.  

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/0d6ab23c-f347-4c8a-b13c-4c8753eaf875.png)

To fix this problem, first make certain that there are no other users issuing checks.  If they are not, then navigate to **System>>Maintenance>>Reset Check Run** and click the **"Reset Cash Account Check Run Flag"** button.   

  
![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/4a23724d-660d-4a8d-a6cb-fdd467488348.png)
