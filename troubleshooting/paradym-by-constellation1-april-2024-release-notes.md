# Paradym by Constellation1 April 2024 Release Notes

Production Release April 2024

## Release Summary  

Y**ouTube Preview Video:** Added download link to email notification.   
**Flyers:** Download link added to property tour email notification.   
**Office Names:** Increased character count for office names.   
**Custom Branding Updates:** Various updated to custom branding.   
**Bug Fixes**

## Release Details  

YouTube Video Notification  
We have added links in the email notifications to download property tour videos once they have been posted on YouTube.   
This new feature provides agents with the ability to easily access and download their branded or unbranded videos using the provided link.

Flyers  
We have simplified the process of creating and downloading property flyers, making it easier to obtain them.  
When a Video Tour is created, a flyer is automatically generated. In the Video Tour notification email, you'll find a download link in the Flyer section for convenient access to download the flyer.

 ![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-ReleaseNotes/Paradym/2024/04%20April/2024-Paradym-DownloadFlyer.jpg)

Office Names

 We’ve updated the character count for office names to 100, allowing brokers to enter the full name of their offices.  

 ![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/KB-ReleaseNotes/Paradym/2024/04%20April/2024-Paradym-OfficeName.jpg)

Custom Branding  
In a recent release, we introduced the option for companies and franchises to apply custom branding to Paradym.   
As part of this release, we are pleased to introduce the following new branding features,  
 Updates include, 

*   Broker emails include custom logo.
*   URL’s in blog posts will be customized to the broker.
*   Click to Chat branded to the broker.
*   Mobile Target URL’s branded to the broker.

Bug Fixes 

1\. We fixed an issue with Paradym not detecting a user’s Facebook Business page.   
Previously, when connecting to Facebook, Paradym was not able to detect the Facebook business page of the user due to a change made by Meta Facebook.  
Now, this issue has been resolved.   
  

2\. We fixed an issue with garage information being removed from Auto Tours.  
Previously, garage information was incorrectly being removed from Auto Tours.  
Now, This issue has been resolved.

  
3\. We fixed an issue with custom logos being uploaded and displayed on some flyers, causing the flyer layout to be malformed.   
Previously, on some flyer templates, when a custom logo is uploaded and displayed on the flyer, other design elements on the flyer would wrap to another line or the flyer would not display with a friendly design.   
Now, this issue has been resolved.
