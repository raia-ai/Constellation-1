# How to Configure QuickBooks® to Run as administrator After Update/Upgrade

Here are the steps required to configure QuickBooks® to Run as administrator after an update or upgrade for non-hosted customers.

After the QuickBooks® update, or upgrade is complete, please ensure both QuickBooks® and Commissions Desktop are closed.

Right click on the QuickBooks® icon and select Properties.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/0db68442-7628-4fa8-b9fc-aee95f25fb73)

On the Compatibility tab, place check to left of Run this program as an administrator

Select Apply and then OK

You can then log into both applications as needed.&#x20;
