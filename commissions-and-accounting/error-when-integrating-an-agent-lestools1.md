# Error When Integrating an Agent > LESTools1

This article is to help you navigate potential error message when adding an Agent in QuickBooks before entering them in Commissions.

You may be prompted when closing a transaction with this error message:

![](https://app.na3.teamsupport.com/Wiki/WikiDocs/2472652/images/QB3.png)

*   This usually occurs when you have entered a new agent into QuickBooks for upfront costs and then added to Commissions.
*   You may need to correct their name in the Customer List, if you had added them and last name.
*   The logic of Commissions is that all customers are entered with Last Name, First name.
*   Once you have confirmed that the period was missing or entered in correctly.
*   In Commissions, go to the agents profile and select Reset Accounting ID
*   Click Okay
*   Click Integrate to accounting
*   Once you see this message you may proceed to the transactions closing
