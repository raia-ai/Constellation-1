# Correcting a Trust Disbursement Check/Cheque

This article details the steps needed to correct an incorrect disbursement.

There may be instances that the trust disbursement was made out to the wrong payee or for the amount was incorrect and a correction is required. 

_**Note: You cannot reverse a disbursement in Commissions.**_ 

**To make the correction, please follow the recommended steps below:** 

*   Delete or Void original check/cheque created in QuickBooks®  
*   Go back to the Trust/Escrow Deposit, select Add New; the action will be Deposit.
*   Enter the same amount and use the same date that was entered for the disbursement 
*   We would suggest to add a description to remind you the reason of add this new deposit.
*   Before saving, ensure that both Post to Accounting and Post Immediately boxes are **unchecked.**

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/18026628)

Once the correcting deposit is saved, you can create a new Disbursement with the correct details, ensuring both Post to Accounting and Post Immediately are checked before you save the correct disbursement.
