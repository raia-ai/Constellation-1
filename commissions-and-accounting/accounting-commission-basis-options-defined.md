# Accounting - Commission Basis Options Defined

The purpose of this article is to explain the different options available for calculating the agent/office split in a sale transaction. 

There are seven commission basis options available for calculating an agent’s commission:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/d5201267-08de-4728-853d-32aef5476330) 

Below you will find screenshots of which field or fields are used for calculating the agent/office split.

**N: Agent Adjusted Gross**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/283e499c-15b6-4be1-8c63-112575e8c7d3) 

**A: Agent after Split**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/2a2e2e4f-d788-47a4-a9e8-5f73e9a7340e) 

**G:  Agent Gross**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/9ec377a6-1333-4d40-b5b6-3aaa9906e2b2) 

**V: Agent Volume**

Sale Price X Side Count = Volume

In the example below, the sale price is $1,000,000 and the agent has both the selling and the buying counts.  The agent’s volume for this sale is calculated as follows:

Sale price $1,000,000 X 2 counts total = $2,000,000 volume.
