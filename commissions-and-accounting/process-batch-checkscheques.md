# Process Batch Checks/Cheques | Constellation1 Customer Hub

After processing multiple transactions for a single agent or Resource, You may only want to print one check this could be done daily, weekly, or monthly. To Batch a single check for multiple transactions please follow the steps below.

*   Navigate to the Accounting / Accounting Batch Processes / Process Batch Check/Cheques
*   Select the applicable From and To Dates to batch
*   Select the applicable Bank Account 
*   Select the applicable Payee Resource and Search 

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/18028163)

*   The list of checks below are all for the same Resource and as you select (check) which ones to include in the batch, the sum will display in RED. 

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/18028164)

*   Once you have selected the ones to batch, select Process.
*   This will void the original checks created (regardless of dates) and “Batch” into one new check. 
*   Below is a sample so you can see how it will be integrated to QuickBooks®  

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/18028165)

**_Note: If checks have been batched you cannot reverse the transaction, as the single check will not be voided through the reversal process._**
