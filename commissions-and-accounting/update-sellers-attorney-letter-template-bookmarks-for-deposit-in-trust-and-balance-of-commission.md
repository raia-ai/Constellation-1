# Update Seller's Attorney Letter Template Bookmarks for \[Deposit in Trust] and \[Balance of Commission

The purpose of this article is to provide the steps required to replace two bookmarks, \[Deposit in Trust] and \[Balance of Commission] on the customized Seller's Attorney Letter ensuring the deposit displays correctly depending if My Company is holding the deposit or not, including the Balance of Commission. &#x20;

To do this, on the left navigation bar select System Settings / Advanced Settings and Templates tab.

Scroll down to the Customized Templates card, and select the download icon to the far right of the customized letter.&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/fabd29a6-a571-4886-8f90-9e9e5df30b1a)

All documents downloaded from Commissions Online will be saved to the Downloads folder of users PC/laptop.

Navigate to the Downloads folder, enable editing if prompted to begin.

Locate the \[Deposit in Trust] field, double click to open the Text Form Field Options window.&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/c61870bf-03cc-4cce-b9b6-1735cdb68ac1)

In the Bookmark field, type les\_trustdeposit. Select OK to close the window.

Next, locate the \[Balance of Commission] field, double click to open the Text Form Field Options window.&#x20;

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/ec303282-f0a6-422b-8e41-bae8a5e2a311)

In the Bookmark field, type les\_balancedue. Select OK to close the window.

Once user has customized the template as needed, they will save as a "**.dotx**" format.&#x20;

To do this, go to File / Save As and by default the location will be **This PC / Documents / Custom Office Templates**.  User should give this template a new name so it is easily identifiable. From the Save as type drop down select Word Template (\*.dotx) and Save.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17637791)

Once saved as a ".dotx", in Commissions Online on the left navigation bar select **System Settings / Advanced Settings** / Templates tab and select + Add Custom Template.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17637792)

Click on Select File button and navigate to the default location of **This PC / Documents / Custom Office Templates** regardless of where user saved it initially.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17637793)

Once user selects the template, click on the green save icon to the far right.

Once you confirm that it is now working as expected, you can delete the previously used template.
