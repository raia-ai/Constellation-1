# Additional System Settings on System Defaults Tab of Company Profile

*   **Prompt Notification When Cancel Button is Selected-** By initiating this flag, user will be prompted when they try to close the system by clicking on the "X" at the top right of the screen. A warning will pop up to confirm that you want to close the system or not, then make your selection.
*   **Enable Update Notifications-** Generally a default setting and will prompt of application update when you initially log in. 
*   **Allow Agent Advance Deduction from Tax Portion-** By placing check in this option, user will be able to include the agents’ portion of Tax as part of the advance repayment upon closing.
*   **Display Date and Time in Fixed Format-** Leave unchecked.
*   **Default Transactions to Not Firm-** Enabled when the transactions are entered with conditions.
*   **Disable SIN/SSN Validation-** By initiating this flag, the software will no longer validate the authenticity of a SIN/SSN (maybe used by brokerages outside North America).
*   **Enable Auto-Naming on Documents-** By initiating this flag, all documents will automatically assign the **INV#** **<YYYY>-<Transaction #>** ex. INV2012\_20120425.doc
*   **Disable Prompt to Update Buyer's Address-** Typically through the process of closing a transaction when you are on the buy side of a transaction Commissions will prompt you to update the buyer with the transaction address. If you do not wish to have that prompt enable this box.

**Agent Profile Search By-** There are three search options available, Agent Legal Name, Agent Business Name, SIN/SSN and your selection will affect how the Agents Profile searches.

*   Agent Legal Name: On Agents Profile, left side ‘Agent ID’ will show Agent’s legal name, and ‘Lookup’ field will search by Agent’s legal name
*   Agent Business Name: On Agents Profile, left side ‘Agent ID’ will show Agent’s Business Name (DBA name), and ‘Lookup’ field will search by Agent’s Business Name
*   Agent SIN/SSN: On Agents Profile, left side ‘Agent ID’ will show Agent’s legal name, and ‘Lookup’ field will search by Agent’s SIN/SSN

*   **Agent Anniversary Reset with New Commission-** Typically set as a default (there is an information icon to explain why users may wish to de-select).
*   **Enable Transaction Audit Trail-** Set as a default and typically a good idea to leave selected as there may be multiple users in the system. This will also enable audit trail tracking
*   **Enable Commission Basis-** Allows users to select various commission calculation basis
*   **Display Agent Name on QuickSearch Window-** Can be selected or deselected per user choice
*   **Adjust Affected Commission When Using Multiple Managers-** When selected, the Agents Detail window, Review Closing and all accounting entries will omit the amounts associated to a Manager Override
*   **Enable Tax on Manager Overrides-** By deselecting, tax will no longer be applied to Manager Override calculations
*   **Enable Leap Years for Interest Calculation Base Days-** This will ensure that if the current year is a leap year, for example, 2020, 2024 and 2028 that interest will be calculated correctly.
*   **Interest Type (Unchecked -- Fixed, Checked -- Variable)-** This will determine the default interest calculation; variable or fixed.

**Deposit Interest Check/Cheque To-** There are three options available and described below: 

*   No Deposit Action: The system will not create any entries to QuickBooks® when the interest check is generated.
*   Deposit to Real Estate Trust: The system will post interest earned and payable to the Buyer and/or other resource to the Real Estate Trust bank account
*   Deposit to Pay Interest from Account: The system will post the interest earned and payable to the Buyer and/or other resource to the appropriate account as set up on the Accounting tab of the Company Profile.
