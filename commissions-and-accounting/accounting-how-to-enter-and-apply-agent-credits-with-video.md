# Accounting - How to Enter and Apply Agent Credits (With Video)

The purpose of this article and video is to provide steps for entering and applying an agent credit.

To enter and apply an agent credit, navigate to **Agents > Processing > Agent Credits/Payments**.

*   Click **Add**.
*   Select **Agent** from the dropdown.
*   **Date Paid** defaults to the current processing date but can be edited.
*   Enter **Amount** as a positive value.
*   Select the GL **Account** to credit from the dropdown.
*   Select **Project** if applicable.
*   Enter **Description**.
    *   **Description** will default to the GL account description but can be edited.
*   Click **Apply Credit/Payment**.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/419870ed-5bd7-4a0c-b2d9-fb279936ad00)

*   Select **Auto-Apply**, or manually enter **Payment** for the charges you would like to credit.
*   Click **OK**.

 ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/db8d5c9e-ef9f-4551-aa06-9f99c8a2e894)

*   Click **Save**
