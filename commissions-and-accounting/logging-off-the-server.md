
# Logging Off The Server | Constellation1 Customer Hub

If Constellation1 Hosts your Commissions application, these are the correct steps to Log Off the server.

*   Please navigate to the Start Menu ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17773055) 

*   Select the Person Icon ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17773056) 

*   Click on Icon and select Sign Out ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/17773057) 

*   You may now log in to see the update has been done.
