# Creating Tasks and Checklists | Constellation1 Customer Hub

This article will provide you with information regarding creating Tasks and Checklists within Commissions Online.Â 

For further information on Managing Tasks and Checklists, see the following.

In Commissions Online, you can assign Tasks and/or Checklists to transactions manually or automatically based on specific criteria.

**To create a Task:**

From the main menu, navigate to System Settings/Checklists, select the Tasks tab and select Add.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/cec47df4-9991-4de9-9fd5-fd758845fdf5)

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/4d98a8a5-9201-40c3-87c4-323234e55818)

At this point, add:

*   **Task Name.**
*   **Task Type,** i.e. Sales Documentation, General Task, etc.
*   Toggle on **Required** if the transaction can NOT be closed until it is completed.
*   Assign a **Task Due Date.**
*   Add any **Notes** you wish to the Task.
*   If desired, toggle on **Enable Notification** and complete the **Notification Recipient** as applicable and should you wish to **Include Primary Transaction Agent**, toggle it on.
*   Assign a Reminder to the Task by selecting **Add Reminde**r and enter the appropriate criteria and select **Save.**

Individual Tasks can be assigned to transactions one at a time. To automate the process, you can create and assign a **Checklist** to a transaction.

**To create a Checklist:**

From the main menu, navigate to **System Settings/Checklists**, select the **Checklist** tab and select **Add.**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/04cc46d4-d367-4cb8-8f8c-f1f277c2a168)

Should you wish the system to Auto-Attach this checklist based on a combination of any or all selected Transaction Type, Property Type and Side, make your appropriate selection and select **Save.**

Your Tasks and Checklists are now ready to be assigned to Transactions!
