# Customizing Letter Templates | Constellation1 Customer Hub

Commissions comes with a wide variety of customizable templates that can be generated through a transaction. Examples would be a 3rd Party Disbursement letter, Letters to Buyers, Sellers, External agents to name a few.

*   Navigate to the Set Up Menu and Letter templates
*   Select the letter to customize
*   When all of customization are complete, save changes, using the floppy disc Icon in the Word Document.
*   Leave the File name as displayed
*   Once the letter template has been customized, you must reselect the letters in your company profile via the Letter Templates node of the System Defaults tab.   This will ensure that all letters will generate as expected with the changes you have made.

**Letters With More Than One Tax Calculation**

Commissions also includes the following templated letters to support a 2nd tax calculation:

*   Advance 2nd Tax.dot
*   Buying Broker 2nd Tax.dot
*   Notary letter.dot (letter to be used when the Buyer's Attorney disburses funds on closing)
*   Seller's Attorney 2nd Tax.dot
*   Selling Broker 2nd Tax.dot

To enable these templates, add them to the Letters node on the System Defaults Tab of the company profile before you can customize them via Letters Templates. Then follow the same process as noted above.
