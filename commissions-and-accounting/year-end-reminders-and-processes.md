
# Year End Reminders and Processes

This article provides information regarding some configuration that should be performed at Year End **OR** on the first business day in the new calendar year, as applicable to your business requirements.

_**Agents:**_ the Anniversary Start Date displayed on the Commission Details will **NEVER** change, however the commission plan will automatically start at Tier 1 using the Anniversary Reset Period selected with the exception of No Limit.  This applies to all other agent Anniversary Reset Period timelines whether it is the first day of each calendar year or another date throughout the year.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/272142bc-a363-482f-b301-baad254a20ba)

**_Commission Plan / Deductions (calendar year caps)_** 

*   From the left navigation bar select **Commissions**
*   Select applicable Commission Plan and **Edit;** or simply click on the plan name
*   Select applicable Deduction and edit the dates accordingly 
*   **Save** the Deduction and Commission Plan

  ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/a6e39fdf-e7dd-4156-a198-96fdf4ceca9d)

**_Edit/Change Transaction Number Prefix and/or Auto Sequence Number_**

*   From the left navigation bar select **System Settings** 
*   Select Advanced Settings / Advanced Settings tab
*   On the Transaction Settings card, enter the new Transaction Number Prefix if required. The Transaction Number Prefix supports dashes and a maximum of 8 characters. This can be used in addition to the Auto Sequence Number to a maximum length of 16 (TR202401-12345678)
*   Auto Sequence Number, enter the applicable first transaction number for the calendar year (maximum of 8 digits)
*   Once completed, select Save
