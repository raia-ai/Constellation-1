
# Receiving a Referral Check | Constellation1 Customer Hub

An agent receives a referral check from another brokerage, for referring a buyer or seller.

When receiving inbound referrals, enter the Property Address and select the Transaction type of Referral.

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/7ca90ff8-7f5c-4140-a470-800e64c23ff2.png)

*   As with any transaction in C1 Commissions, in order to successfully process a transaction all parties need to be entered, regardless of whether you have that information.
*   Select a Buyer/Seller (If unknown, use a placeholder) ie. Unknown Seller/ Buyer.
*   Select the Override (OV) option on the Gross Commission field and enter the Referral amount.
*   No sales price is required for a referral received.
*   Select the co-brokerage who submitted the Referral.
*   Complete the transaction process as normal.
