# Creating Flat Rate Commission Plans

This document will guide the user through the process to create a new Flat Rate Commission Plan manually.

**Note**: The Commission Wizard is enabled by default. To create commission plans manually, the wizard can be disabled in System Settings / Advanced Settings.

To create a plan:

*   Select Commissions from the left hand navigation bar.
*   Select the **Add+** button.

On the first screen,

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/56997183-7e58-408a-96e5-38114febc360)

*   Assign a name for the new plan.
*   Select Flat Rate as the Plan Type.
*   Expand the Advanced Options section and make any selections for Calculate Adjusted Company Portion, toggle on if this is a team plan and whether you wish to prorate across tiers.
*   In the Value field, enter the percentage being paid to the agent **OR** select the percentage button to change to a dollar value should this commission plan pay a flat dollar amount.
*   Click on Add Deduction button should you wish to assign a deduction to the commission plan and select the edit button to configure the deduction.
*   Make your selections as to:
    *   Deduct From and Calculation Basis.
    *   Whether it applies to Non-Commission Income.
    *   If it is being split across Agents, or;
    *   If it is being pro-rated across tiers.
*   Under Commission Plan Tiers assign a percentage and/or a dollar amount and choose whether the value of this deduction is the Sum of the two, smaller or larger amount.
*   If this deduction has a cap, toggle this on and assign an amount and the cap date and configure the date range should it be Custom.
*   If this deduction has an additional amount that applies should the cap be reached, assign a percentage and/or a dollar amount and choose whether the value of this deduction is the Sum of the two, smaller or larger amount and select **Save.**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/263e0fba-859c-41a8-9de8-c1deacdadfe7)

*   Select **Save** and the Flat Rate Commission Plan will display in the list on the Commissions/Commission Plans tab.
