# Accounting - How to change background color from white to grayscale inverted

The purpose of this article is provide steps to personalize the color theme for your desktop and Accounting program.  

Below is a screenshot of the Accounting program, which by default has a white background.  

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/6cb06251-ecf2-45f9-bc4f-3f11319c7886.png)

If you find this background to be too bright, you may want to try changing the color theme to grayscale inverted to see if that is easier on your eyes.  Below is a screenshot of the Accounting program, with grayscale inverted selected in Windows Personalization.  

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/1352bb3d-cad5-4e0b-9945-7c5946c31fc8.png)

To change the color theme to grayscale inverted:

*   Right-Click on Desktop
*   Select **Personalize**

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/253350ea-81b5-4b23-9ec7-b549ad606d13.png)

*   Select **Themes**

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/1cb60fa3-d34b-40d5-bbab-9aaced512f65.png)

*   Select **High contrast settings**

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/141086af-cc84-427e-a9c5-0a7727208a0d.png)

*   Select **Color Filters**

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/b071aedf-34f4-4791-986c-932cdf1124a8.png)

*   **Turn on color filters** and select **Grayscale inverted**

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/1b0e9c6d-03c7-4197-a118-5083ad3315d6.png)
