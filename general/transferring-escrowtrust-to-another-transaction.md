# Transferring Escrow/Trust to Another Transaction

There are situations where a Buyer has placed a deposit on certain property that fell through, but the Buyer purchases another property and requested that Deposit continue to be held by your brokerage.

*   In this instance, navigate to the original transaction and select Edit.
*   Navigate to the Deposits tab and select Escrow/Trust icon.
*   Highlight the deposit, then select the Transfer icon:

 ![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/d5cca223-a1df-4385-a719-191328a48f63.png)

*   You will prompted a notification advising this operation will reverse the original Deposit and reassign it to a new transaction. If you have already reconciled the original you will then need to re-reconcile.

**To Attach the Deposit to Another Transaction**

*   Navigate to the new transaction and select Edit
*   Navigate to the Deposits tab and select Escrow/Trust icon.
*   Highlight the appropriate deposit and select Attach at the top right of the window:

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/ce0827d2-72ba-4a2c-bc76-d8b89845894a.png)

*   Highlight the appropriate deposit and select Attach on the Transferred Trust window.
*   A notification will display advising that this deposit will be attached to this transaction and removed from the transfer list, select OK.

NOTE: This will not automatically update the original deposit with the new transaction number or address in QuickBooks® and should be edited manually in QuickBooks® as required.
