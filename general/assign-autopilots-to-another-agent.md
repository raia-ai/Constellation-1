# Assign Autopilots to Another Agent

*   In order to assign Autopilots to another agent click on the “OFFICES/AGENTS” page.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/9ebfd5b6-f035-4c2b-a192-6a0f47a2fb07)

*   Search the agent by name in the search bar of whom you want to change the Autopilots for another agent and select “Edit” by clicking on 3 dots … under Actions column.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/19f132a8-64dd-423e-abdf-f0109c43a566)

*   In the Edit section click on the “Referrals” tab and select the Autopilots you want to change the Agent for and click on the “Change Agent” button.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/7a6b10a9-b725-4c0b-939c-7c5c5ed3442c)

*   A new modal box will be opened from which you can select a new agent by clicking on the dropdown menu, to whom you want to assign the selected Autopilots.

                                  ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/7662d3d5-42ea-42a3-b3ff-c4672265782b)

*   Assign a new agent from the drop-down menu and click on “Change” button and the Autopilot record(s) will be updated.

                                ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/034f31bb-e929-44fd-941e-2486f8103a2e)

**Note:** If an agent has left, you can also assign his/ her autopilots to the active agent by following the above-given procedure.
