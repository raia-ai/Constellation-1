# REALedger - How to edit the default percentage amount of a pre-split deduction globally.

The purpose of this article is to show how to edit the default percentage amount of a pre-split deduction globally.

To edit the default percentage of a pre-split deduction, first navigate to **System>>Maintenance>>Offices**.  Then follow the steps below:

*   Edit **Default PCT Amount**
*   **Save**

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/20275454)

*   Navigate to **System>>Maintenance>>Bulk Update Defaults in Agent and Pending Files**
*   Select pre-split from the **Deduction Code** dropdown
*   Enter new percentage amount in the **Default PCT Amount** field
*   Check the box below the **Default PCT Amount**
*   Check the box below **Recalculate Pendings?** 
*   Click **Update Pre-Split** button
