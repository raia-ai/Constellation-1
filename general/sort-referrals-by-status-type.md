# Sort referrals by Status Type

*   To filter the referral record with different statuses, click on the “Clients” page, select the “Date Range” and then click on “Filter”.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/71e46e24-6bcc-493d-be84-f3a10bbb29be)

*   Select the required “Status” and click the "Search" button.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/a090e7df-e555-4fdb-8f68-51d218cfcfa1)

*   List of all the records with the selected “Status” types will be displayed. You can download the list by clicking on "Export Data”.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/60bc0c22-84da-49e8-aed4-06d215fdecf9)
