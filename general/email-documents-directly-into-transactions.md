# Email documents directly into Transactions

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/1407ba92-99f4-4751-bd88-d21315909d1b)

*   Email the documents to \[Transaction-ID\]@trpoint.com
*   Transaction IDs can be looked up by logging in to you Transactions account, and looking up the relevant Transaction ID under the "Transaction #" field as shown below:
*   ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/6a33eebe-4479-43ed-994a-1670a14c821d)
*   In the screenshot above, If a document needs to be emailed directly to the first transaction in that list, then it would be emailed to "911055-909085@trpoint.com"
