# 1/10/2025 - Upcoming System Conversion for Royal Gorge Association of REALTORS (RGAR) Data Feed

We want to inform you of an upcoming change that may affect your use of the **Royal Gorge Association of REALTORS (RGAR)** data feed.

On **Friday January 10th, 2025**, a vendor system conversion from **Navica** to **PikesPeak (vendor system)** will be implemented for this data feed. You can access this data by adding the **$reconvert=true** flag in your queries. Please note that there is no change to the OSN that you are currently using.

**Key values in all resources will be changing (ListingKey, MemberKey, OfficeKey, etc.)** as a result of this conversion.
