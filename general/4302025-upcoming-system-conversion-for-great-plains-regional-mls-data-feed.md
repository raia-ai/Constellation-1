# 4/30/2025 - Upcoming System Conversion for Great Plains Regional MLS Data Feed

Please find attached an **updated Agent/Office ID Translation Document**, which reflects the latest corrections.

As a result of these updates, the conversion timeline has been extended and will now be completed on **Wednesday, April 30, 2025**.

You can begin accessing the updated data by including the **$reconvert=true** flag in your API queries. Please note that there is no change to the OriginatingSystemName (OSN) you are currently using.

With this conversion, key values in all resources will be changing (ListingKey, MemberKey, OfficeKey, etc.)
