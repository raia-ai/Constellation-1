# 12/11/2024 - Upcoming System Conversion for Southern Missouri Regional MLS Data Feed

On Wednesday December 11th, 2024, a vendor system conversion will be implemented for this data feed. To ensure a smooth transition, we’ve made the updated data available in advance. You can access this data by adding the **$reconvert=true** flag in your queries. 

Please note that **both Office and Member IDs will be changing**. To facilitate a seamless transition for your operations, we have attached detailed information outlining the upcoming changes. Also, **k****ey values in all resources will be changing** (ListingKey, MemberKey, OfficeKey, etc.) as a result of this conversion.
