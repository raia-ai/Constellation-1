# Search for Duplicates | Constellation1 Customer Hub

**Search for Duplicates**

 In the Clients section, press “Add” and select either “Quick Outgoing” or “Quick Incoming”:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/e60a798c-0d82-4488-b106-fab554fda7bf)

The “Search for duplicates” function works exclusively based on “Last Name” or “First Name” fields. You will be prompted to the input page where you will see the following message:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/927e4660-063a-49e8-8208-9179cf6a69da)

Type in the last name and press “Check Duplicate”:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/df05c83d-082d-4aab-9d7e-ff140440d649)

The following pop-up will be displayed:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/e3a55364-3d31-444d-b66f-2137c2bcdda2)

Press the eye button (circled in red) to view the existing record:

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/5c95ca1a-34f6-4cbd-8aa4-e3b79989d647)
