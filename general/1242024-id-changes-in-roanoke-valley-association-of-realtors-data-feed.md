# 1/24/2024 - ID Changes in Roanoke Valley Association of REALTORS Data Feed

We would like to notify you about an upcoming change in the Roanoke Valley Association of REALTORS data feed scheduled for Wednesday, January 24th, 2024. Attached herewith is a comprehensive file detailing the anticipated ID modifications that will be reflected in this data feed.

The following fields within their respective entities will be affected:

**Property Resource:**

*   ListAgentMlsId
*   CoListAgentMlsId
*   BuyerAgentMlsId
*   CoBuyerAgentMlsId
*   ListOfficeMlsId
*   CoListOfficeMlsId
*   BuyerOfficeMlsId
*   CoBuyerOfficeMlsId

**Member Resource:**

*   MemberMlsId
*   AltMemberMlsId
*   OfficeMlsId

**Office Resource:**

*   OfficeMlsId
*   AltOfficeMlsId
