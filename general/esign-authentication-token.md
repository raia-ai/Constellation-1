# eSign - Authentication Token | Constellation1 Customer Hub

### Authentication Token

Authentication in the eSign web service is token based. The API key is encrypted and sent to the server which returns a token to be used with subsequent web service calls. To ensure security, an authentication token has a very short life and should be refreshed prior to every API request.
