
# Select Region for Agents | Constellation1 Customer Hub

*   In order to select a region for any Agent you first need to add a region by clicking on "Preferences" under the drop-down icon having the name of your account.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/502cc65e-a6a7-46e0-9a81-ec541569ab68)

*   Scroll down the page and click on the “Agent Regions” button.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/a7df0ad5-c8bf-4a97-8278-ff475e500e38)

*   Add a new region by clicking on “Add Agent Region”. Type the region name

in the “New Tag” field and submit it by clicking on “Save”.

*   A newly added region can be selected for any agent by clicking on

“OFFICES/AGENTS” page and by adding/editing an agent.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/335ddea6-5ef2-4b1a-8c4c-6cee675c53f6)

*   In order to select a region for a new agent you need to click on

“Add -> Agent”.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/212ec090-1b50-4648-8f83-5a54684a1dad)

*   Fill in all the required fields, select a region from the “Region” drop-down list and submit by clicking on the “Save” button.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/6cae6a69-dba9-40b8-bec4-0e9b053a4692)

*   In order to select a region for an existing Agent you need to be on the “OFFICES/AGENTS” page, click on … dots under the Action column and then select “Edit”.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/c9e66dab-c50b-4598-9268-917f299de083)

*   Select a region from the “Region” drop-down list and click on the “Save” button.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/1145e67d-92a9-4a56-a510-00873ea4b291)
