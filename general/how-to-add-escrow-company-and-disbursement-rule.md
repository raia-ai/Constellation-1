# How to Add Escrow Company and Disbursement Rule

There may be an instance where you may need to change the way the funds are being disbursed by Title or Escrow company, and a different rule is required.

*   Navigate to Transaction and select EDIT 
*   On the Deposits tab, expand the panel/node labeled “ESCROW” and complete as required 
*   Select the Resource then SEARCH 

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/7145a97b-ee3b-4847-b407-2f4d815c8df3.png)

*   Lookup name OR ADD 
*   Once highlighted, select OK 
*   Follow the same steps to select the Disbursement Rule 

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/b2dda441-90cd-4cee-8d11-0271c9a95a2b.png)
