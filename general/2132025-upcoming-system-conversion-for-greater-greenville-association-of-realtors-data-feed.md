# 2/13/2025 - Upcoming System Conversion for Greater Greenville Association of REALTORS Data Feed

On **Thursday February 13th, 2025**, a vendor system conversion from RETS to webAPI will be implemented for this data feed. To ensure a smooth transition, we’ve made the updated data available in advance. You can access this data by adding the **$reconvert=true** flag in your queries.

As a result of this conversion, **both Office and Member IDs will be changing**. we have attached detailed information outlining the upcoming changes. **Key values in all resources will be changing (ListingKey, MemberKey, OfficeKey, etc.)** as well.
