# eSign - REST-Update Signers | Constellation1 Customer Hub

### Update Signers

URI PUT rest/v1/signers

**Returns** – A UpdateSignersView containing the result of updated signers. (See UpdateSignersView in Appendix B)

**Parameters**

Sample JSON looks like this:  
{  
“username”: “name@email.com”,  
“signingSessionId”: “some GUID”,  
“signers”: \[..\]  
}
