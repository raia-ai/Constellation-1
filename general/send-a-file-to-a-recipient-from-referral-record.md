
# Send a File to a Recipient from Referral Record

*   In order to send a file directly from a Referral record click on the “Clients” page.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/e8a2eb4e-e832-4496-9511-7fd8fcfc342b)

*   Search the referral record by typing the name of the client in the search bar and selecting the “Edit” option by clicking on 3 dots … under the Actions column.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/80827cf7-d5a6-472b-94fb-debb1b45a658)

*   Click on the “Files” section and select the “Send” option by clicking on 3 dots … under the Action column.  
     

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/f277b146-6548-4a0d-ac4d-33f7e0d57cc3)

*   A new modal box will appear once you click on the “Send” option. Fill in the fields and then click on the “Send” button.
