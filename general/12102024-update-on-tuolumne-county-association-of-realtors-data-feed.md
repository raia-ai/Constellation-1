# 12/10/2024 - Update on Tuolumne County Association of REALTORS Data Feed

We would like to inform you that the data previously associated with the Tuolumne County Association of REALTORS data feed (OSN Tuolumne) has been migrated and is now accessible through the bridgeMLS data feed (OSN BridgeMLS).

To locate these listings, you can reference the ListOfficeAOR field in bridgeMLS, which will display the value "Tuolumne County AOR".
