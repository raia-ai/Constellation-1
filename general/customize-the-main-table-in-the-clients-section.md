# Customize the Main Table in the “Clients” section

**Customize the Main Table in the “Clients” section**

Go to the “Clients” section. Click “Select Columns” to see the options available:

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/2ac7cb4d-c5a7-47e0-8035-78d955048a24.png)

Activate or de-activate the necessary columns by simply clicking on them:

![](https://constellation1.na3.teamsupport.com/api/attachments/image/2472652/669a6e5b-4e92-4c2d-9627-1e68bc241901.png)

Arrange the table based on your preferences: drag & drop column headers from left to right. Remember, the first 3 columns can’t be moved. The rest is customizable!
