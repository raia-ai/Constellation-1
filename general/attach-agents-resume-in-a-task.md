# Attach Agent’s Resume in a task

*   To add an agent’s resume to a task click on the “Clients“ page.

                                                                             ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/57328199-38a4-4bd5-b55c-5e776067c67a)

*   On the client's page, please enter the name of the client for whom you wish to make this change in the search bar designated for this purpose. Upon entering the name, click on the corresponding entry to edit it.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/ddb840f6-4efd-4ac0-b067-6cab27f78b50)

*   Now click on the “Tasks” tab and select the “Add task” option.

  ![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/31763964-b110-4930-bc67-a7d3a471509c)

*   A dialog box will be opened, fill in the fields as per your requirement and type “Resume- Assigned Agent” in the Uploads field before sending it to the recipient.

.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/fbf0f8bb-fb7b-4cbe-98cc-2fb3f0c58ddc)

*   The recipient on the other end will receive the Agent’s resume file as an attachment of the task email.

![](https://constellation1.na3.teamsupport.com/api/attachments/action/1/d49a3f4a-b1df-4169-9094-fca74436d6bb)
