# 12/5/2024 - Upcoming System Conversion for Metro MLS Data Feed

We would like to notify you of an upcoming change that may impact your use of the Metro MLS data feed.

On Thursday December 5th, 2024, a vendor system conversion will be implemented for this data feed. To ensure a smooth transition, we’ve made the updated data available in advance. You can access this data by adding the **$reconvert=true** flag in your queries. Please note that there is no change to the OSN that you are currently using.

If you have any questions or need further assistance, please don't hesitate to reach out.
