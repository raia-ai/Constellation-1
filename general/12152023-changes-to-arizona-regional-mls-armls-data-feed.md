# 12/15/2023 - Changes to Arizona Regional MLS (ARMLS) data feed

We would like to inform you about an upcoming change that will impact your experience with the Arizona Regional Multiple Listing Service (ARMLS).

On December 15th, 2023, a vendor system conversion will take place, necessitating a shift to the new OSN: ArizonaRegional. It is imperative to note that following this conversion, the current feed using OSN ARMLS will become stale.
