# Service Tiles BOI Files | Constellation1 Customer Hub

The Service Tiles BOI file creates records that will have these entries appear on core service pages. These records get associated to agents via the agent boi file, and they get associated to offices via the office boi file.

Here are the fields that get sent over in this file:

Action|FirstName|LastName|InternalID|OfficeInternalID|IsListedInAgentFinder|Active|DirectOfficePhone|MobilePhone|Email|CompanyName|SecondaryOfficeInternalIDs|LogoUrl|JobTitle|WebsiteUrl|GetAQuoteUrl|PrequalifyUrl|CalculatorUrl|GetInfoUrl|FindAVacationRentalUrl
